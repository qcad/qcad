1. Seleccione las entidades que desea proyectar.
2. Inicie esta herramienta.
3. Seleccione un tipo de proyección y una dirección de visualización para la
 proyección en la barra de herramientas de opciones.
4. Establezca el punto de referencia para la proyección. Este es el punto
 que usará para posicionar la proyección en el siguiente paso.
5. Mueva el cursor del ratón a la posición en la que desea crear la
 proyección y haga clic para fijar la posición.
