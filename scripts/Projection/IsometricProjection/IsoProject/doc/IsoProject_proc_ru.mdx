1. Выберите объекты, которые хотите спроецировать.
2. Запустите этот инструмент.
3. Выберите тип проекции и направление обзора для проекции на панели опций.
4. Задайте опорную точку для проекции. Это точка, которую вы будете использовать
   для позиционирования проекции на следующем шаге.
5. Переместите курсор мыши в положение, где хотите создать проекцию, и щёлкните,
   чтобы задать положение.
