1. Виберіть об'єкти, які ви хочете спроектувати.
2. Запустіть цей інструмент.
3. Виберіть тип проекції та напрямок огляду для проекції на панелі параметрів.
4. Установіть опорну точку для проекції. Це точка, яку ви використовуватимете для позиціонування проекції на наступному кроці.
5. Перемістіть курсор миші в положення, де ви хочете створити проекцію, і клацніть, щоб установити положення.
