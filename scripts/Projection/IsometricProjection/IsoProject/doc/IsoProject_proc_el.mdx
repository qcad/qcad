1. Επιλέξτε τις οντότητες που θέλετε να προβάλετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Επιλέξτε έναν τύπο προβολής και μια κατεύθυνση θέασης για την προβολή στη
   γραμμή εργαλείων επιλογών.
4. Ορίστε το σημείο αναφοράς για την προβολή. Αυτό είναι το σημείο που θα
   χρησιμοποιήσετε για να τοποθετήσετε την προβολή στο επόμενο βήμα.
5. Μετακινήστε τον δρομέα του ποντικιού στη θέση όπου θέλετε να δημιουργήσετε την
   προβολή και κάντε κλικ για να ορίσετε τη θέση.
