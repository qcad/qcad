1. Utilice las herramientas de selección para seleccionar una o varias entidades cuyas propiedades desee ver o cambiar.
2. Las propiedades de la selección se muestran en el editor de propiedades.

Si se selecciona una sola entidad, se muestran todas sus propiedades, incluida su geometría (por ejemplo, el punto inicial y final de una línea o el centro y el radio de un círculo). Si se seleccionan varias entidades, el editor de propiedades muestra las propiedades comunes a todas las entidades seleccionadas. Una propiedad con valores diferentes en la selección se muestra como un campo mixto.

Para cambiar una propiedad, introduzca un nuevo valor en el campo correspondiente del editor de propiedades y confirme su entrada (Intro o Tab).

El cambio se aplica inmediatamente a todas las entidades seleccionadas. De esta forma se pueden editar las propiedades geométricas, los atributos como capa, color, grosor de línea y tipo de línea, así como las propiedades personalizadas.

Si no hay nada seleccionado, el editor de propiedades está vacío.
