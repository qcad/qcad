1. Utilizzare gli strumenti di selezione per selezionare una o più entità di cui si desidera visualizzare o modificare le proprietà.
2. Le proprietà della selezione vengono visualizzate nell'editor delle proprietà.

Se è selezionata una singola entità, vengono mostrate tutte le sue proprietà, inclusa la sua geometria (ad esempio il punto iniziale e finale di una linea o il centro e il raggio di un cerchio). Se sono selezionate più entità, l'editor delle proprietà mostra le proprietà comuni a tutte le entità selezionate. Una proprietà con valori diversi nella selezione viene visualizzata come campo misto.

Per modificare una proprietà, inserire un nuovo valore nel campo corrispondente dell'editor delle proprietà e confermare l'immissione (Invio o Tab).

La modifica viene applicata immediatamente a tutte le entità selezionate. In questo modo è possibile modificare le proprietà geometriche, gli attributi come livello, colore, spessore linea e tipo di linea, nonché le proprietà personalizzate.

Se non è selezionato nulla, l'editor delle proprietà è vuoto.
