1. Use the selection tools to select one or multiple entities whose
   properties you want to view or change.
2. The properties of the selection are displayed in the property editor.

If a single entity is selected, all of its properties are shown, including
its geometry (for example the start and end point of a line or the center
and radius of a circle). If multiple entities are selected, the property
editor shows the properties that are common to all selected entities. A
property with different values across the selection is displayed as a mixed field.

To change a property, enter a new value into the corresponding field in the
property editor and confirm your input (Enter or Tab). 

The change is immediately applied to all selected entities. Geometric properties, 
attributes such as layer, color, line weight and line type, as well as custom 
properties can be edited this way.

If nothing is selected, the property editor is empty.
