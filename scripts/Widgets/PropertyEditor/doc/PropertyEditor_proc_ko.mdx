1. 선택 도구를 사용하여 속성을 보거나 변경하려는 하나 이상의 개체를 선택합니다.
2. 선택 항목의 속성이 속성 편집기에 표시됩니다.

단일 개체가 선택된 경우 형상(예: 선의 시작점과 끝점 또는 원의 중심과 반지름)을 포함한 모든 속성이 표시됩니다. 여러 개체가 선택된 경우 속성 편집기에는 선택한 모든 개체에 공통된 속성이 표시됩니다. 선택 항목에서 값이 서로 다른 속성은 혼합 필드로 표시됩니다.

속성을 변경하려면 속성 편집기의 해당 필드에 새 값을 입력하고 입력을 확인합니다(Enter 또는 Tab).

변경 사항은 선택한 모든 개체에 즉시 적용됩니다. 이 방법으로 기하학적 속성, 레이어·색상·선 두께·선종류와 같은 속성, 그리고 사용자 지정 속성을 편집할 수 있습니다.

선택된 항목이 없으면 속성 편집기는 비어 있습니다.
