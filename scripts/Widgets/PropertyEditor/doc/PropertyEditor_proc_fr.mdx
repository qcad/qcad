1. Utilisez les outils de sélection pour sélectionner une ou plusieurs entités dont vous souhaitez afficher ou modifier les propriétés.
2. Les propriétés de la sélection s'affichent dans l'éditeur de propriétés.

Si une seule entité est sélectionnée, toutes ses propriétés sont affichées, y compris sa géométrie (par exemple le point de départ et d'arrivée d'une ligne ou le centre et le rayon d'un cercle). Si plusieurs entités sont sélectionnées, l'éditeur de propriétés affiche les propriétés communes à toutes les entités sélectionnées. Une propriété ayant des valeurs différentes selon la sélection est affichée sous forme de champ mixte.

Pour modifier une propriété, saisissez une nouvelle valeur dans le champ correspondant de l'éditeur de propriétés et confirmez votre saisie (Entrée ou Tab).

La modification est immédiatement appliquée à toutes les entités sélectionnées. Les propriétés géométriques, les attributs tels que le calque, la couleur, l'épaisseur de ligne et le type de ligne, ainsi que les propriétés personnalisées peuvent être modifiés de cette manière.

Si rien n'est sélectionné, l'éditeur de propriétés est vide.
