Η γραμμή εντολών του QCAD σάς επιτρέπει να εκκινείτε εντολές, να εισάγετε συντεταγμένες ή
να εισάγετε τιμές όπως αποστάσεις ή ακτίνες.

### Εισαγωγή συντεταγμένων

Οι απόλυτες συντεταγμένες εισάγονται στη μορφή «x,y»:

40,5

Οι σχετικές συντεταγμένες εισάγονται στη μορφή «@x,y»:

@10,6

Οι απόλυτες πολικές συντεταγμένες εισάγονται στη μορφή «απόσταση&lt;γωνία»:

10&lt;30

Οι σχετικές πολικές συντεταγμένες εισάγονται στη μορφή «@απόσταση&lt;γωνία»:

@10&lt;45

Κατά την εισαγωγή συντεταγμένων ή τιμών, μπορούν να χρησιμοποιηθούν [μαθηματικές εκφράσεις](#expr) για τον υπολογισμό μιας συντεταγμένης βάσει γνωστών τιμών. Για παράδειγμα η
συντεταγμένη 10,5 μπορεί επίσης να εισαχθεί ως:

5+5,30/6

### Αριθμομηχανή

Η γραμμή εντολών μπορεί επίσης να χρησιμοποιηθεί ως αριθμομηχανή. Για να το κάνετε αυτό, εισαγάγετε μια [μαθηματική έκφραση](#expr) με προηγούμενο ένα σύμβολο ίσον:

=3+4  
7

Μπορούν να χρησιμοποιηθούν μεταβλητές για την αποθήκευση τιμών:

=a=5+6  
11  
=a/2  
5.5  

### Μαθηματικές εκφράσεις

Οι διαθέσιμες μαθηματικές σταθερές είναι:

PI, LN2, LN10, LOG2E, LOG10E, SQRT1\_2, SQRT2

Οι διαθέσιμες μαθηματικές συναρτήσεις είναι:

abs, ceil, floor, exp, log, max, min, pow, sqrt, random, round, rad2deg, deg2rad,
sin, cos, tan, asin, acos, atan, atan2, log10, log1p, log2, sign, cosh, sinh, tanh,
acosh, asinh, atanh, expm1, hypot, cbrt, trunc

Οι περισσότερες από αυτές τις σταθερές και συναρτήσεις είναι τυπικές συναρτήσεις ECMAScript (JavaScript) και
τεκμηριώνονται διαδικτυακά. Στο τυπικό ECMAScript, αυτές οι συναρτήσεις αποτελούν μέρος της κλάσης Math, οπότε η
συνάρτηση *abs* θα έπρεπε να γραφτεί ως *Math.abs*. Στη γραμμή εντολών του QCAD, μπορείτε να
παραλείψετε το τμήμα *Math.* για ευκολία. Οι τριγωνομετρικές συναρτήσεις (sin, cos, tan, asin, acos, atan, atan2)
δέχονται ή επιστρέφουν γωνίες σε μοίρες. Αν προτιμάτε τις εκδόσεις σε ακτίνια αυτών των συναρτήσεων,
χρησιμοποιήστε αντ' αυτού τις αρχικές συναρτήσεις *Math.*.

Οι συναρτήσεις *rad2deg* και *deg2rad* μπορούν να χρησιμοποιηθούν για τη μετατροπή γωνιών μεταξύ
ακτινίων και μοιρών.
