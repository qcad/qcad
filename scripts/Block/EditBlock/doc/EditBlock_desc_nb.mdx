Dette verktøyet åpner den aktive blokken for redigering. Den kan endres akkurat
som hovedtegningen. For å gå tilbake til hovedtegningen (kalt "*Model_Space"),
aktiver blokken "*Model_Space" og klikk på den samme knappen igjen for å redigere
den blokken, eller start verktøyet «Rediger hovedtegning» i menyen «Blokk».
