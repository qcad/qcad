1. Scegliere il blocco da inserire dalla lista dei blocchi.
2. Fare click sul pulsante di inserimento o scegliere "Inserisci blocco" dal
 menu.
3. Inserire l'angolo di rotazione e il fattore di scala per il riferimento
 di blocco dalla barra delle opzioni.
4. Specifica la posizione del blocco inserito cliccando su una coordinata o
 inserendo una coordinata nella console.
5. Per creare un intero array di blocchi, fare clic sul pulsante array nella barra degli strumenti delle opzioni:  
![](ArrayOfBlocks.png)  
Inserire le colonne, le righe, la spaziatura delle colonne e la spaziatura delle righe nella finestra di dialogo visualizzata.
 Queste proprietà possono essere modificate successivamente utilizzando l'editor delle proprietà.
