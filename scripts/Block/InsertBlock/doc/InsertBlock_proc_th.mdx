1. เลือกบล็อกที่คุณต้องการแทรกจากรายการบล็อก
2. คลิกปุ่มแทรกหรือเลือก „Insert Block" จากเมนู
3. ป้อนมุมหมุนและตัวคูณสเกลสำหรับการอ้างอิงบล็อกในแถบเครื่องมือตัวเลือก
4. ระบุตำแหน่งของบล็อกที่แทรกโดยการคลิกที่พิกัดหรือป้อนพิกัดในคอนโซล
5. หากต้องการสร้างอาร์เรย์ของบล็อกทั้งหมด ให้คลิกปุ่มอาร์เรย์ในแถบเครื่องมือตัวเลือก:  
   <Image src={ArrayOfBlocks} alt="ArrayOfBlocks" width={40} height={40} />  
   ป้อนจำนวนคอลัมน์ แถว ระยะห่างคอลัมน์ และระยะห่างแถวในกล่องโต้ตอบที่แสดงขึ้น คุณสมบัติเหล่านี้สามารถแก้ไขได้ในภายหลังโดยใช้ตัวแก้ไขคุณสมบัติ
