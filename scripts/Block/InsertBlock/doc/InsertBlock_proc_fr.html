1. Sélectionnez le bloc que vous voulez insérer dans la liste des blocs.
  2. Cliquez sur le bouton 'Insérer' ou choisissez 'Insérer un bloc' dans le menu.
  3. Entrer l'angle de rotation et le facteur d'échelle pour la référence du bloc dans la barre d'outils des options.
  4. Spécifiez la position du bloc inséré en cliquant sur une coordonnée ou en entrant une coordonnée dans la console.
  5. Pour créer un tableau complet de blocs, cliquez sur le bouton de tableau dans la barre d'outils des options:

    Entrez les colonnes, les lignes, l'espacement des colonnes et l'espacement des lignes dans la boîte de dialogue affichée. Ces propriétés peuvent également être modifiées ultérieurement à l'aide de l'éditeur de propriétés.
