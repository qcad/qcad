import ArrayOfBlocks from './ArrayOfBlocks.png';

1. Επιλέξτε το μπλοκ που θέλετε να εισαγάγετε από τη λίστα μπλοκ.
2. Κάντε κλικ στο κουμπί εισαγωγής ή επιλέξτε «Εισαγωγή μπλοκ» από το μενού.
3. Εισαγάγετε τη γωνία περιστροφής και τον συντελεστή κλίμακας για την αναφορά μπλοκ στη
   γραμμή εργαλείων επιλογών.
4. Ορίστε τη θέση του εισαγόμενου μπλοκ κάνοντας κλικ σε μια συντεταγμένη ή
   εισάγοντας μια συντεταγμένη στην κονσόλα.
5. Για να δημιουργήσετε έναν ολόκληρο πίνακα μπλοκ, κάντε κλικ στο κουμπί πίνακα στη γραμμή εργαλείων επιλογών:  
   <Image src={ArrayOfBlocks} alt="ArrayOfBlocks" width={40} height={40} />  
   Εισαγάγετε τις στήλες, τις σειρές, την απόσταση στηλών και την απόσταση σειρών στον διάλογο που εμφανίζεται.
   Αυτές οι ιδιότητες μπορούν επίσης να επεξεργαστούν αργότερα χρησιμοποιώντας τον επεξεργαστή ιδιοτήτων.
