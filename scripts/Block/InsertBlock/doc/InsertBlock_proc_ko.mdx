import ArrayOfBlocks from './ArrayOfBlocks.png';

1. 블록 목록에서 삽입할 블록을 선택합니다.
2. 삽입 버튼을 클릭하거나 메뉴에서 „블록 삽입”을 선택합니다.
3. 옵션 도구 모음에서 블록 참조의 회전 각도와 축척 비율을
   입력합니다.
4. 좌표를 클릭하거나 콘솔에 좌표를 입력하여 삽입된 블록의
   위치를 지정합니다.
5. 블록 전체 배열을 만들려면 옵션 도구 모음에서 배열 버튼을 클릭합니다:  
   <Image src={ArrayOfBlocks} alt="ArrayOfBlocks" width={40} height={40} />  
   표시된 대화 상자에서 열, 행, 열 간격, 행 간격을 입력합니다.
   이러한 속성은 나중에 속성 편집기를 사용하여 편집할 수도 있습니다.
