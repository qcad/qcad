1. Pilih blok yang ingin Anda sisipkan dari daftar blok.
2. Klik tombol Sisipkan atau pilih „Sisipkan Blok" dari menu.
3. Masukkan sudut rotasi dan faktor skala untuk referensi blok di bilah
 alat opsi.
4. Tentukan posisi blok yang disisipkan dengan mengklik koordinat atau
 memasukkan koordinat di konsol.
5. Untuk membuat seluruh array blok, klik tombol array di bilah alat opsi:  
![](ArrayOfBlocks.png)  
Masukkan kolom, baris, jarak kolom dan jarak baris dalam dialog yang ditampilkan.
 Properti ini juga dapat diedit nanti menggunakan editor properti.
