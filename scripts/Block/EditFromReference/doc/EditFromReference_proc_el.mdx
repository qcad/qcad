- Κάντε κλικ σε οποιαδήποτε οντότητα αναφοράς μπλοκ που αναφέρεται στο μπλοκ που θέλετε να
  επεξεργαστείτε.

Το μπλοκ ανοίγει για επεξεργασία. Για να επιστρέψετε στο κύριο σχέδιο (που ονομάζεται
 "*Model_Space"), επεξεργαστείτε το μπλοκ "*Model_Space" ή ξεκινήστε το εργαλείο "Επεξεργασία κύριου
 σχεδίου" στο μενού "Μπλοκ".
