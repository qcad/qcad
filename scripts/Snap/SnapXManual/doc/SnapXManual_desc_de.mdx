Dieses Werkzeug kann benutzt werden, um eine Koordinate (z.B. X) von einem
Objekt zu übernehmen und die andere (Y) von einem anderen Objekt.  
Dies kann zum Beispiel benutzt werden, um das Zentrum eines Rechtecks zu
finden.  
Dieses Konzept wird auch als "Kombinieren von Koordinatenwerten" oder
"Koordinatenfilter" bezeichnet.
