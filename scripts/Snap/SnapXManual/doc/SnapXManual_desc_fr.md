Cet outil peut être utilisé pour extraire une coordonnée (par exemple X) de
l'emplacement d'une entité existante et l'autre coordonnée (par exemple Y) de
l'emplacement d'un autre objet.  
Ceci peut par exemple être utilisé pour localiser le centre d'un
rectangle.  
Ce concept est également appelé " combinaison de coordonnées ", "
filtrage de points " ou " filtrage de coordonnées ".
