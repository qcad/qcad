Esta herramienta de ajuste se puede utilizar para extraer una coordenada
(por ejemplo, X) de la ubicación de una entidad existente y la otra coordenada
(por ejemplo, Y) de la ubicación de otro objeto.  
Esto se puede utilizar, por ejemplo, para localizar el centro de un
rectángulo.  
Este concepto se denomina también "combinación de coordenadas", "filtrado
de puntos" o "filtrado de coordenadas".
