Цей інструмент прив'язки можна використовувати для вилучення однієї координати (наприклад X) з розташування одного наявного об'єкта та іншої координати (наприклад Y) з розташування іншого об'єкта.  
Це можна, наприклад, використати для визначення центру прямокутника.  
Ця концепція також називається „комбінуванням координат", „фільтрацією точок" або „фільтрацією координат".
