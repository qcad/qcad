Этот инструмент привязки можно использовать для извлечения одной координаты
(например, X) из местоположения одной существующей сущности и другой координаты
(например, Y) из местоположения другого объекта.  
Это может быть использовано, например, для определения центра
прямоугольника.  
Эту концепцию также называют "комбинированием координат", "фильтрацией
точек" или "фильтрацией координат".
