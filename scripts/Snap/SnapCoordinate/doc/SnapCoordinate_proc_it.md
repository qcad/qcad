1. Avviare questo strumento quando si deve specificare un punto e si
 desidera farlo inserendo una coppia di coordinate (x/y).
2. Inserire le coordinate nella barra delle opzioni e scegliere se si tratta
 di coordinate assolute o relative (relative al punto zero relativo).
3. Fare clic sul pulsante OK o premere Invio per confermare l'immissione e
 impostare le coordinate:
