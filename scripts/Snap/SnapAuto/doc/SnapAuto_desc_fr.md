L'outil d'accrochage auto s'enclenche sur le point d'intersection, le point
final, le point médian, le point perpendiculaire, le point de référence, le
point de grille ou le point le plus proche sur une entité dans cet ordre de
priorité.  
Les types de points de référence auxquels l'outil d'accrochage auto peut
être configuré sous *Edition > Préférences générales > Accrochage > Accrochage
auto*
