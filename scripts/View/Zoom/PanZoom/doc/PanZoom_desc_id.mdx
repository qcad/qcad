Panning berarti bergerak (menggulir) di sekitar gambar. Cara tercepat untuk melakukannya adalah menggunakan tombol tengah mouse dan menggerakkan tampilan serupa dengan cara Anda menggerakkan kertas: tekan tombol tengah mouse dan tahan sambil menggerakkan gambar. Jika mouse Anda tidak memiliki tombol tengah, Anda dapat menekan tombol Control (Mac OS X: tombol Command) dan menggunakan tombol kiri mouse sebagai gantinya.  
Sebagai alternatif, Anda dapat mengaktifkan alat ini dan kemudian menggerakkan tampilan dengan tombol kiri mouse tanpa menekan tombol apa pun.  
Klik tombol kanan mouse saat Anda selesai.
