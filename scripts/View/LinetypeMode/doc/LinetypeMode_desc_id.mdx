Mengalihkan mode tipe garis berbasis layar gambar saat ini.  
Jika mode ini diaktifkan, tipe garis dioptimalkan untuk layar komputer. Semua garis ditampilkan dengan lebar garis berbasis layar (garis tidak menjadi lebih lebar saat memperbesar) dan semua pola ditampilkan sebagai pola tetap berbasis piksel di layar (garis putus-putus tidak menjadi lebih panjang saat memperbesar).  
Jika mode ini nonaktif, lebar garis dan tipe garis ditampilkan dalam satuan gambar (default).
