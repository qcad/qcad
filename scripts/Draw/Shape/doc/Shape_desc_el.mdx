Τα σχήματα είναι κλειστές γεωμετρικές φιγούρες όπως ορθογώνια ή κανονικά πολύγωνα.

#### Κοινές επιλογές σχήματος

*Πολυγραμμή*

Τα εργαλεία σχήματος μπορούν είτε να δημιουργήσουν χαλαρές γραμμές και τόξα είτε μια κλειστή πολυγραμμή.
 Επιλέξτε «Δημιουργία πολυγραμμής» για να δημιουργήσετε μία μόνο οντότητα πολυγραμμής ανά κατασκευασμένο σχήμα.

*Γέμισμα*

Επιλέξτε το πλαίσιο ελέγχου «Γέμισμα» για να δημιουργήσετε ένα στερεό γέμισμα επιπλέον του περιγράμματος του σχήματος.

*Ακτίνα*

Επιλέξτε το πλαίσιο ελέγχου «Ακτίνα» για να στρογγυλέψετε όλες τις γωνίες του σχήματος με τη δοθείσα ακτίνα.
