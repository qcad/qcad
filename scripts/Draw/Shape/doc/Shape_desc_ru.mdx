Фигуры представляют собой замкнутые геометрические объекты, такие как прямоугольники или правильные многоугольники.

#### Общие параметры фигур

*Полилиния*

Инструменты для создания фигур могут создавать либо свободные линии и дуги, либо замкнутую полилинию. Отметьте "Создать полилинию", чтобы создать одну
 полилинию для каждой построенной фигуры.

*Заливка*

Установите флажок "Заливка", чтобы создать сплошную заливку в дополнение к границам фигуры.

*Радиус*

Установите флажок "Радиус", чтобы скруглить все углы фигуры с заданным радиусом.
