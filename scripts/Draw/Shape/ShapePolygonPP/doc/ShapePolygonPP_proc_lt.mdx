1. Parinkčių įrankių juostoje pasirinkite norimas bendras [figūros parinktis](/lt/manual/scripts/Draw/Shape/doc/Shape).
2. Parinkčių įrankių juostoje įveskite daugiakampio laipsnį (kraštinių / kampų skaičių). Galiojantys skaičiai yra nuo 3 iki 99.
3. Nustatykite pirmą daugiakampio kampą pele arba įveskite koordinatę komandų eilutėje.
4. Nustatykite antrą daugiakampio kampą.
