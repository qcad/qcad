1. Valitse haluamasi yleiset [muotovaihtoehdot](/fi/manual/scripts/Draw/Shape/doc/Shape) asetustyökalurivillä.
2. Syötä monikulmion sivujen lukumäärä (reunat / kulmat) asetustyökaluriville. Kelvolliset luvut ovat välillä 3–99.
3. Aseta monikulmion ensimmäinen kulma hiirellä tai syötä koordinaatti komentoriville.
4. Aseta monikulmion toinen kulma.
