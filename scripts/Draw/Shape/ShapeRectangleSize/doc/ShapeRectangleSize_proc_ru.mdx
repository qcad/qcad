1. Выберите нужные [параметры](/ru/manual/scripts/Draw/Shape/doc/Shape) общей формы на панели инструментов.
2. Введите размер прямоугольника на панели инструментов.
3. Выберите опорную точку, которую нужно использовать для позиционирования прямоугольника.
4. Укажите положение прямоугольника с помощью мыши или введя координаты в командной строке.
