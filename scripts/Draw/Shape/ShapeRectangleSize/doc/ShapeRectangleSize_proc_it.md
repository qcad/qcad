1. Scegliere le [opzioni di forma](/it/manual/scripts/Draw/Shape/doc/Shape) comune desiderate nella barra degli strumenti delle opzioni.
2. Immettere le dimensioni del rettangolo nella barra degli strumenti delle opzioni.
3. Scegliere il punto di riferimento da utilizzare per posizionare il rettangolo.
4. Specificate la posizione del rettangolo con il mouse o inserendo una coordinata nella riga di comando.
