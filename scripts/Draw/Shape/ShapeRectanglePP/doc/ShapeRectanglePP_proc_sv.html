1. Välj önskat alternativ för den gemensamma formen i verktygsfältet för alternativ.
  2. Välj önskat
  3. Ange rektangelns första hörn.
  4. Flytta musen till det andra hörnet och klicka för att ange rektangelns andra hörn. Alternativt kan du ange koordinaten för det andra hörnet i kommandotolken. Om du t.ex. vill skapa en rektangel med bredden 50 och höjden 25 anger du den relativa koordinaten för det andra hörnet på följande sätt:
    @50,25
