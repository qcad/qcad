1. Pilih [opsi bentuk](/id/manual/scripts/Draw/Shape/doc/Shape) umum yang diinginkan pada bilah alat opsi.
2. Tentukan sudut pertama persegi panjang.
3. Gerakkan mouse ke sudut kedua dan klik untuk menentukan sudut kedua persegi panjang. Sebagai alternatif, Anda dapat memasukkan koordinat sudut kedua pada prompt perintah. Mis. untuk membuat persegi panjang dengan lebar 50 dan tinggi 25, masukkan koordinat relatif sudut kedua sebagai berikut:  
   @50,25
