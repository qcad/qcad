1. 옵션 도구 모음에서 원하는 공통 [형태 옵션](/ko/manual/scripts/Draw/Shape/doc/Shape)을 선택합니다.
2. 옵션 도구 모음에 다각형의 차수(변 / 모서리 개수)를
   입력합니다. 유효한 숫자 범위는 3에서 99까지입니다.
3. 두 개의 반대쪽 모서리로 다각형을 정의하려면 모서리에서 모서리로
   옵션을 선택합니다.
4. 마우스를 사용하여 다각형의 첫 번째 모서리 또는 변의 중간을
   설정하거나 명령줄에 좌표를 입력합니다.
5. 다각형의 두 번째, 반대쪽 모서리 또는 변의 중간을 설정합니다.
