import HatchFromSelectionBoundary from './HatchFromSelectionBoundary.png';
import HatchFromSelectionIslands from './HatchFromSelectionIslands.png';

1. 해치 영역을 둘러싸는 개체가 닫힌 윤곽을 형성하도록 준비합니다.
   윤곽은 이 예의 오른쪽에 표시된 것처럼 한 개체가 다음 개체에
   연결되는 방식으로 닫혀 있어야 합니다:  
   <Image src={HatchFromSelectionBoundary} alt="HatchFromSelectionBoundary" width={130} height={76} />
2. 채우려는 윤곽을 선택합니다. 윤곽 내부의 섬은 선택하지 않으면
   해치된다는 점에 유의하십시오:  
   <Image src={HatchFromSelectionIslands} alt="HatchFromSelectionIslands" width={105} height={45} />
3. 해치 도구를 시작합니다.
4. 해치 옵션 대화 상자가 표시됩니다. 해치 패턴, 축척 비율, 해치
   패턴의 회전 각도를 선택합니다. 패턴 대신 솔리드 색상으로 개체를
   채우려면 „솔리드 채우기” 확인란을
   선택합니다.
5. „OK”를 클릭하여 해치를 진행합니다. 윤곽의 복잡성과 선택한 패턴의
   축척 비율에 따라 해치를 만드는 데 시간이 다소 걸릴 수
   있습니다.
