1. Tentukan lokasi fitur dengan mouse atau dengan memasukkan koordinat.
2. Tentukan lokasi label teks dan titik akhir garis penunjuk. Jika
   garis penunjuk lebih dekat ke vertikal, datum X ditunjukkan, jika lebih
   dekat ke horizontal, datum Y ditunjukkan.
3. Titik asal dimensi ordinat berada pada titik nol absolut. Untuk
   memindahkan titik asal dengan mouse, pilih entitas dimensi ordinat dan
   pindahkan titik asal dengan mouse (seret dan lepas). Untuk menentukan
   koordinat untuk titik asal, gunakan editor properti.
