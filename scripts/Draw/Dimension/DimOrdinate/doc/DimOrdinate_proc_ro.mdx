1. Specificați locația elementului cu mouse-ul sau introducând o coordonată.
2. Specificați locația etichetei de text și punctul final al indicatorului. Dacă indicatorul este mai aproape de vertical, se indică referința X, dacă este mai aproape de orizontal, se indică referința Y.
3. Originea cotei ordonate este în punctul zero absolut. Pentru a muta originea cu mouse-ul, selectați entitatea cotei ordonate și mutați originea cu mouse-ul (trageți și plasați). Pentru a specifica o coordonată pentru origine, utilizați editorul de proprietăți.
