1. 在選項工具列中選擇雲形線的階數。支援的階數為 2（二次 b 雲形線）和 3（三次 b 雲形線）。階數越高，曲線越「平滑」。  
   請注意，根據定義，二次 b 雲形線至少需要 3 個控制點，而三次 b 雲形線至少有 4 個控制點。
2. 如果要建立封閉雲形線，請在選項工具列中勾選「Closed」核取方塊。封閉雲形線是連續的閉環。
3. 指定控制點。定義第三個（對於二次 b 雲形線）或第四個（對於三次 b 雲形線）控制點後，將顯示預覽。您可以再次按一下選項工具列中的「Undo」按鈕來移除最後一個控制點。
4. 設定最後一個點後，按 Escape 或按一下滑鼠右鍵以確認雲形線形狀。
5. 現在您可以建立另一個雲形線，或再次右鍵以終止工具。
