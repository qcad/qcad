1. 옵션 도구 모음에서 스플라인의 차수를 선택합니다. 지원되는 차수는
   2(2차 b-스플라인)와 3(3차 b-스플라인)입니다. 차수가 높을수록
   곡선이 더 „부드러워”집니다.  
   2차 b-스플라인은 정의상 최소 3개의 제어점이 필요하고 3차 b-스플라인은
   최소 4개의 제어점을 가진다는 점에 유의하십시오.
2. 닫힌 스플라인을 만들려면 옵션 도구 모음에서 „닫힘” 확인란을
   선택합니다. 닫힌 스플라인은 연속적인 닫힌 루프입니다.
3. 제어점을 지정합니다. 세 번째(2차 b-스플라인의 경우) 또는 네
   번째(3차 b-스플라인의 경우) 제어점을 정의한 후 미리 보기가
   표시됩니다. 옵션 도구 모음에서 „실행 취소” 버튼을 클릭하여 마지막
   제어점을 다시 제거할 수 있습니다.
4. 마지막 점을 설정한 후 Escape를 누르거나 마우스 오른쪽 버튼을 클릭하여
   스플라인 모양을 확정합니다.
5. 이제 다른 스플라인을 만들거나 다시 마우스 오른쪽 버튼을 클릭하여
   도구를 종료할 수 있습니다.
