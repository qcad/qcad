1. เลือกกำลังของสไปลน์ในแถบเครื่องมือตัวเลือก กำลังที่รองรับคือ 2 (b-spline กำลังสอง) และ 3 (b-spline กำลังสาม) ยิ่งกำลังสูง เส้นโค้งก็จะ „ราบรื่น" มากขึ้น  
   โปรดทราบว่า b-spline กำลังสองต้องการจุดควบคุมอย่างน้อย 3 จุดตามคำจำกัดความ ในขณะที่ b-spline กำลังสามมีจุดควบคุมอย่างน้อย 4 จุด
2. ทำเครื่องหมายที่ช่อง „Closed" ในแถบเครื่องมือตัวเลือกหากคุณต้องการสร้างสไปลน์ปิด สไปลน์ปิดคือวงปิดต่อเนื่อง
3. ระบุจุดควบคุม หลังจากกำหนดจุดควบคุมที่สาม (สำหรับ b-spline กำลังสอง) หรือที่สี่ (สำหรับ b-spline กำลังสาม) ตัวอย่างจะแสดงขึ้น คุณสามารถลบจุดควบคุมสุดท้ายได้อีกครั้งโดยการคลิกปุ่ม „Undo" ในแถบเครื่องมือตัวเลือก
4. กด Escape หรือคลิกปุ่มขวาของเมาส์หลังจากกำหนดจุดสุดท้ายเพื่อยืนยันรูปทรงสไปลน์
5. ตอนนี้คุณสามารถสร้างสไปลน์อีกอันหรือคลิกขวาอีกครั้งเพื่อสิ้นสุดเครื่องมือ
