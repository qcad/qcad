1. Επιλέξτε τον βαθμό της spline στη γραμμή εργαλείων επιλογών. Οι υποστηριζόμενοι
   βαθμοί είναι 2 (τετραγωνική b-spline) και 3 (κυβική b-spline). Όσο υψηλότερος ο
   βαθμός, τόσο πιο «ομαλή» γίνεται η καμπύλη.  
   Σημειώστε ότι μια τετραγωνική b-spline απαιτεί εξ ορισμού τουλάχιστον 3
   σημεία ελέγχου, ενώ μια κυβική b-spline έχει τουλάχιστον 4 σημεία ελέγχου.
2. Επιλέξτε το πλαίσιο ελέγχου «Κλειστή» στη γραμμή εργαλείων επιλογών αν θέλετε να
   δημιουργήσετε μια κλειστή spline. Οι κλειστές spline είναι συνεχείς κλειστοί βρόχοι.
3. Ορίστε τα σημεία ελέγχου. Μετά τον ορισμό του τρίτου (για τετραγωνικές
   b-splines) ή τέταρτου (για κυβικές b-splines) σημείου ελέγχου, θα εμφανιστεί μια
   προεπισκόπηση. Μπορείτε να αφαιρέσετε ξανά το τελευταίο σημείο ελέγχου κάνοντας κλικ στο κουμπί
   «Αναίρεση» στη γραμμή εργαλείων επιλογών.
4. Πατήστε Escape ή κάντε κλικ στο δεξί κουμπί του ποντικιού αφού ορίσετε το τελευταίο σημείο
   για να επιβεβαιώσετε το σχήμα της spline.
5. Μπορείτε τώρα να δημιουργήσετε μια άλλη spline ή να κάνετε ξανά δεξί κλικ για να τερματίσετε το
   εργαλείο.
