1. Scegliere il grado di scanalatura nella barra degli strumenti delle
   opzioni.  
   Si noti che una b-spline quadratica richiede per definizione almeno 3
   punti di controllo mentre una b-spline cubica ha almeno 4 punti di
   controllo.
2. Selezionare la casella di controllo 'Chiuso' nella barra degli
   strumenti delle opzioni se si desidera creare una scanalatura chiusa. Le
   scanalature chiuse sono a ciclo chiuso continuo.
3. Specificare i punti di controllo.
4. Premete il tasto destro del mouse dopo aver impostato l'ultimo punto
   per confermare la forma della scanalatura.
5. È possibile creare un'altra scanalatura o fare nuovamente clic con
   il pulsante destro del mouse per terminare l'operazione.
