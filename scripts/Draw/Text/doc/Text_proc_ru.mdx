1. После запуска инструмента отображается диалог текста, в котором вы можете
   выбрать шрифт и ввести текст, который хотите создать.
2. Выберите название шрифта, высоту текста и начальные атрибуты текста в
   верхней левой части диалога.  
   Шрифты CAD, состоящие из линий и дуг, перечислены вверху списка шрифтов,
   шрифты TrueType — ниже.  
   В поле «Высота» введите высоту текста в единицах вашего чертежа. Чтобы
   выбрать обычный межстрочный интервал, введите коэффициент межстрочного
   интервала или «1». Вы можете увеличить или уменьшить межстрочный интервал,
   введя другое значение. Введённое значение является коэффициентом
   относительно значения по умолчанию (1.0).
3. Введите угол поворота текста. Обычные горизонтальные тексты имеют угол
   0 градусов, а тексты, читаемые справа, — 90 градусов.
4. Выберите выравнивание текста.
5. Установите флажок «Простой текст», чтобы создать однострочный текст без
   какого-либо расширенного форматирования.
6. Введите текст, который хотите создать, в большом текстовом поле.  
   Вы можете использовать различные инструменты над полем для дальнейшего
   форматирования введённого текста или для вставки текстов из буфера обмена.
   Используйте кнопку справа, чтобы быстро вставить символы или выбрать символ
   Юникода («Другие...»).  
   Обратите внимание, что выбранный вами шрифт может не содержать вставляемые
   вами символы. В этом случае символы не будут отображаться на чертеже.
7. Нажмите «OK», чтобы выйти из диалога текста.
8. С помощью мыши укажите расположение текста или введите координату в
   командной строке.
9. Часто пользователям нужно создать несколько похожих текстов. По этой причине
   инструмент текста не завершается после создания первого текстового объекта.
   Вы можете изменить текст и угол наклона текста на панели опций и создать
   другие тексты с похожими настройками.
10. Щёлкните правой кнопкой мыши или нажмите Escape, чтобы остановить инструмент
    текста.
