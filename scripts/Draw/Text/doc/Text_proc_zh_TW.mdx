1. 啟動工具後，將顯示文字對話方塊，您可以在其中選擇字型並輸入要建立的文字。
2. 在對話方塊左上角區域選擇字型名稱、文字高度和初始文字屬性。  
   由線和弧組成的 CAD 字型列在字型列表頂部，TrueType 字型列在下方。  
   在「Height」欄位中，以您圖紙的單位輸入文字的高度。要選擇正常行距，請輸入行距因子或「1」。您可以通過輸入替代值來增大或減小行距。輸入的值是相對於預設值 (1.0) 的因子。
3. 輸入文字的旋轉角度。正常的水平文字角度為 0 度，從右側可讀的文字角度為 90 度。
4. 選擇文字的對齊方式。
5. 勾選「Simple Text」核取方塊以建立沒有任何進階格式的單行文字。
6. 在大文字框中輸入要建立的文字。  
   您可以使用框上方的各種工具來進一步格式化輸入的文字，或從剪貼簿貼上文字。使用右側的按鈕快速插入符號或選擇 Unicode 字元（「Others...」）。  
   請注意，您選擇的字型可能不包含您插入的字元。在這種情況下，這些字元將不會在圖紙中顯示。
7. 按一下「OK」退出文字對話方塊。
8. 用滑鼠指定文字的位置，或在指令行中輸入座標。
9. 使用者通常需要建立多個相似的文字。因此，文字工具在建立第一個文字實體後不會終止。您可以在選項工具列中更改文字和文字角度，並使用相似的設定建立其他文字。
10. 右鍵或按 Escape 停止文字工具。
