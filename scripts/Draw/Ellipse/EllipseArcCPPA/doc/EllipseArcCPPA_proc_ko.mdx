1. 마우스를 사용하여 타원의 중심을 설정하거나 명령줄에 좌표를
   입력합니다.
2. 타원 위의 점인 축의 끝점을 클릭하여 장축을 정의합니다. 명령줄에
   좌표를 입력하거나 @50&lt;30 형식으로 각도와 장반경을 입력할 수도
   있습니다. 여기서 50은 장반경이고 30은 타원
   각도입니다.
3. 역시 타원 위의 점인 단축의 끝점을 정의하거나 단축의 길이를
   입력합니다.
4. 마우스를 사용하거나 명령줄에 좌표 또는 각도를 입력하여 시작
   각도를 설정합니다.
5. 시작 각도와 동일한 방식으로 끝 각도를 설정합니다.
