1. Pilih arah busur pada bilah alat opsi.
2. Atur pusat busur menggunakan mouse atau masukkan koordinat pada
   baris perintah.
3. Tentukan radius dengan mengklik titik pada busur atau dengan memasukkan
   koordinat pada baris perintah. Anda juga dapat memasukkan radius pada
   baris perintah.
4. Atur sudut awal dengan mouse atau dengan memasukkan koordinat atau sudut
   pada baris perintah.
5. Atur sudut akhir dengan cara yang sama seperti sudut awal.
