Αυτό το εργαλείο σάς επιτρέπει να σχεδιάζετε πολυγραμμές. Μια πολυγραμμή είναι μια συνεχής γραμμή που σχηματίζεται
από ένα ή περισσότερα συνδεδεμένα τμήματα γραμμής ή τόξου.  
Σημειώστε ότι είναι σχεδόν πάντα πιο βολικό να χρησιμοποιείτε τα κανονικά
εργαλεία γραμμής και τόξου για να σχεδιάσετε μια νέα πολυγραμμή. Τα χαλαρά συνδεδεμένα τμήματα γραμμής και τόξου
μπορούν στη συνέχεια να μετατραπούν σε οντότητα πολυγραμμής χρησιμοποιώντας το εργαλείο «Δημιουργία
πολυγραμμής από τμήματα».
