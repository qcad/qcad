1. Tentukan titik awal segmen polyline pertama. Anda dapat menggunakan
   mouse atau memasukkan koordinat pada konsol.
2. Tentukan titik akhir segmen polyline pertama.
3. Tentukan titik akhir segmen polyline tambahan.

Jika polyline berisi segmen busur, Anda dapat mencentang kotak centang "Arc"
dan memasukkan radius busur pada bilah alat opsi saat mengatur titik akhir
segmen busur. Busur akan terhubung secara mulus (tangensial) ke segmen
terakhir.
