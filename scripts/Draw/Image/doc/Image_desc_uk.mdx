Вставляє растрове зображення (бітову карту) в креслення.  
Зауважте, що файл креслення, який створюється під час збереження креслення, містить лише посилання на вставлені зображення. Рекомендується зберігати файл зображення та файл креслення в одній папці, щоб QCAD міг знову знайти зображення, коли файл креслення буде завантажено пізніше.  
Прозорий фон підтримується для зображень PNG.  
Зауважте, що великі бітові карти можуть спричинити дуже повільне відображення креслення.  
Зазвичай ви захочете, щоб зображення були на задньому плані інших об'єктів. Зверніться до інструмента ['Modify' - 'Send to back'](/uk/manual/scripts/Modify/DrawOrder/ToBack/doc/ToBack), щоб дізнатися, як це зробити.
