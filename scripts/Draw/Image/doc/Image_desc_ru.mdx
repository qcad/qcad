Вставляет растровое изображение (bitmap) в чертёж.  
Обратите внимание, что файл чертежа, создаваемый при сохранении чертежа,
содержит только ссылки на вставленные изображения. Рекомендуется хранить файл
изображения и файл чертежа в одной папке, чтобы QCAD мог снова найти
изображение при последующей загрузке файла чертежа.  
Для изображений PNG поддерживается прозрачный фон.  
Обратите внимание, что большие растровые изображения могут сильно замедлить
отображение чертежа.  
Обычно изображения нужно размещать позади других объектов. Обратитесь к
инструменту [«Изменить» — «На задний план»](/ru/manual/scripts/Modify/DrawOrder/ToBack/doc/ToBack),
чтобы узнать, как это сделать.
