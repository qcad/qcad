1. Επιλέξτε το αρχείο εικόνας που θέλετε να εισαγάγετε.
2. Εισαγάγετε το επιθυμητό ύψος ή πλάτος για την εικόνα σε μονάδες σχεδίου στη
   γραμμή εργαλείων επιλογών.
3. Πληκτρολογήστε μια γωνία περιστροφής στη γραμμή εργαλείων επιλογών.
4. Ορίστε το σημείο εισαγωγής της εικόνας. Αυτό είναι η κάτω αριστερή γωνία
   της εικόνας.
