Fügt ein Raster Bild (Bitmap) in die Zeichnung ein.  
Beachten Sie, dass die Zeichnungsdatei, die beim Speichern erstellt wird,
lediglich Referenzen auf Bilddateien enthält. Es ist empfehlenswert, die
Bilddateien und die Zeichnungsdatei im gleichen Ordner zu speichern.  
Transparente Pixel werden für das PNG Format unterstützt.  
Beachten Sie, dass große Bilder das Anzeigen der Zeichnung stark
verlangsamen können.  
Normalerweise ist es wünschenswert, dass Bilder hinter anderen Objekten
dargestellt werden. Bitte beachten Sie dazu das ['Modifizieren' - 'In den Hintergrund'](/de/manual/scripts/Modify/DrawOrder/ToBack/doc/ToBack) Werkzeug.
