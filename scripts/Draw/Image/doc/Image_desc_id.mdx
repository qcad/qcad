Menyisipkan citra raster (bitmap) ke dalam gambar.  
Perhatikan bahwa file gambar yang dibuat saat menyimpan gambar hanya berisi
referensi ke citra yang disisipkan. Disarankan untuk menyimpan file citra
dan file gambar dalam folder yang sama, sehingga QCAD dapat menemukan citra
kembali saat file gambar dimuat nanti.  
Latar belakang transparan didukung untuk citra PNG.  
Perhatikan bahwa bitmap besar dapat menyebabkan tampilan gambar menjadi
sangat lambat.  
Biasanya Anda ingin citra berada di latar belakang entitas lain. Silakan
lihat alat ['Modify' - 'Send to back'](/id/manual/scripts/Modify/DrawOrder/ToBack/doc/ToBack)
untuk mempelajari cara melakukannya.
