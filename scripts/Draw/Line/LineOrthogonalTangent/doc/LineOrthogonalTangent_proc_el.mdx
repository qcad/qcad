1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Κάντε κλικ στην οντότητα γραμμής προς την οποία θα πρέπει να είναι ορθογώνια η νέα γραμμή.
3. Κάντε κλικ στην οντότητα τόξου, κύκλου ή έλλειψης στην οποία θα πρέπει να είναι εφαπτόμενη η
   γραμμή. Υπάρχουν δύο πιθανές λύσεις. Καθώς μετακινείτε το ποντίκι, μπορείτε
   να δείτε μια προεπισκόπηση της εφαπτομένης που θα δημιουργηθεί.
