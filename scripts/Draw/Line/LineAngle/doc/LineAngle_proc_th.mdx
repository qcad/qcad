1. เลือก [ประเภทเส้น](/th/manual/scripts/Draw/Line/doc/Line#type) ที่ต้องการในแถบเครื่องมือตัวเลือก
2. ป้อนมุมและความยาวของเส้นในแถบเครื่องมือตัวเลือก
3. ในแถบเครื่องมือตัวเลือก ให้เลือกจุดอ้างอิงบนเส้นที่คุณต้องการใช้สำหรับการวางตำแหน่งเส้น „Start" หมายความว่าเส้นจะมีจุดเริ่มต้นอยู่ที่จุดที่คุณวางตำแหน่ง
4. วางเส้นด้วยเมาส์หรือโดยการป้อนพิกัดในบรรทัดคำสั่ง
