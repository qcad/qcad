1. Choose the desired
[line type](/en/manual/scripts/Draw/Line/doc/Line#type) in the options
toolbar.
2. Enter the angle and length of the line in the options tool bar.
3. In the options tool bar, choose the reference point on the line which you
want to use for positioning the line. 'Start' means that the line will have its
start point at the point where you position it.
4. Place the line with the mouse or by entering a coordinate in the command
line.
