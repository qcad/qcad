1. Wählen Sie den gewünschten
   [Linientyp](/de/manual/scripts/Draw/Line/doc/Line#type) in der
   Optionenwerkzeugleiste.
2. Geben Sie den Winkel und die Länge der Linie in der Optionenwerkzeugleiste
   ein.
3. Wählen Sie den Referenzpunkt mit dem Sie die Linie platzieren möchten in
   der Optionenwerkzeugleiste. 'Start' bedeutet, dass die Linie dort startet wo
   Sie sie positionieren.
4. Positionieren Sie die Linie mit der Maus oder geben Sie eine Koordinate in
   die Kommandozeile ein.
