1. Välj önskad
   [linjetyp](/sv/manual/scripts/Draw/Line/doc/Line#type) i verktygsfältet för
   alternativ.
2. Ange linjens vinkel och längd i verktygsfältet för alternativ.
3. Välj den referenspunkt på linjen som du vill använda för att positionera
   linjen i verktygsfältet för alternativ. "Start" innebär att linjen kommer att
   ha sin startpunkt på den punkt där du positionerar den.
4. Placera linjen med musen eller genom att ange en koordinat på
   kommandoraden.
