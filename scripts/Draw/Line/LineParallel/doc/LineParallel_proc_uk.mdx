1. Виберіть потрібний [тип лінії](/uk/manual/scripts/Draw/Line/doc/Line#type) на панелі параметрів.
2. Введіть відстань концентричного або паралельного об'єкта від початкового об'єкта на панелі параметрів, показаній угорі.
3. Введіть кількість паралельних або концентричних об'єктів для створення на панелі параметрів.
4. Клацніть базовий об'єкт. Паралельні або концентричні об'єкти створюються на тому боці, на якому розташований курсор миші під час вибору об'єкта.
