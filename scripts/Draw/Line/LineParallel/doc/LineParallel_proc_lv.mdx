1. Opciju rīkjoslā izvēlieties vēlamo [līnijas veidu](/lv/manual/scripts/Draw/Line/doc/Line#type).
2. Augšpusē redzamajā opciju rīkjoslā ievadiet koncentriskās vai paralēlās vienības attālumu no sākotnējās vienības.
3. Opciju rīkjoslā ievadiet izveidojamo paralēlo vai koncentrisko vienību skaitu.
4. Noklikšķiniet uz bāzes vienības. Paralēlās vai koncentriskās vienības tiek izveidotas tajā pusē, kurā, atlasot vienību, atrodas peles kursors.
