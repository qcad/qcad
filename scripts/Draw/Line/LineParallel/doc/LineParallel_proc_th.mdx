1. เลือก [ประเภทเส้น](/th/manual/scripts/Draw/Line/doc/Line#type) ที่ต้องการในแถบเครื่องมือตัวเลือก
2. ป้อนระยะห่างของเอนทิตีร่วมศูนย์กลางหรือขนานจากเอนทิตีเดิมในแถบเครื่องมือตัวเลือกที่แสดงด้านบน
3. ป้อนจำนวนเอนทิตีขนานหรือร่วมศูนย์กลางที่จะสร้างในแถบเครื่องมือตัวเลือก
4. คลิกที่เอนทิตีฐาน เอนทิตีขนานหรือร่วมศูนย์กลางจะถูกสร้างขึ้นในด้านที่เคอร์เซอร์เมาส์อยู่ขณะเลือกเอนทิตี
