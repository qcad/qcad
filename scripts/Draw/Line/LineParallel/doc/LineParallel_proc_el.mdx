1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Εισαγάγετε την απόσταση της ομόκεντρης ή παράλληλης οντότητας από την αρχική
   οντότητα στη γραμμή εργαλείων επιλογών που εμφανίζεται στο επάνω μέρος.
3. Εισαγάγετε τον αριθμό των παράλληλων ή ομόκεντρων οντοτήτων που θα δημιουργηθούν στη
   γραμμή εργαλείων επιλογών.
4. Κάντε κλικ στην οντότητα βάσης. Οι παράλληλες ή ομόκεντρες οντότητες δημιουργούνται στην
   πλευρά στην οποία βρίσκεται ο δρομέας του ποντικιού κατά την επιλογή της οντότητας.
