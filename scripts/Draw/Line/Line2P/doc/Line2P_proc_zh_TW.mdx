import Close from './Close.png';
import Undo from './Undo.png';

1. 在選項工具列中選擇所需的[線型](/zh_TW/manual/scripts/Draw/Line/doc/Line#type)。
2. 指定第一條線段的起點。您可以使用滑鼠或在控制台中輸入座標。
3. 指定第一條線段的端點。
4. 指定其他線段的端點。按一下選項工具列中的「Close」按鈕以閉合序列：
   <Image src={Close} alt="Close" width={40} height={40} />  
   如果您需要復原單條線段，可以按一下「Undo」按鈕：
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. 您可以使用選項工具列中的相應輸入框將直線的長度或角度設定為固定值。
