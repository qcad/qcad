import Close from './Close.png';
import Undo from './Undo.png';

1. Velg ønsket [linjetype](/nb/manual/scripts/Draw/Line/doc/Line#type) i verktøylinjen for alternativer.
2. Angi startpunktet for det første linjesegmentet. Du kan bruke musen eller skrive inn en koordinat i konsollen.
3. Angi endepunktet for det første linjesegmentet.
4. Angi endepunktene for ytterligere linjesegmenter. Klikk på «Lukk»-knappen i verktøylinjen for alternativer for å lukke sekvensen:<br/>
   <Image src={Close} alt="Close" width={40} height={40} />  
   Hvis du trenger å angre et enkelt linjesegment, kan du gjøre det ved å klikke på «Angre»-knappen:<br/>
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Du kan sette lengden eller vinkelen til linjen til faste verdier ved hjelp av de aktuelle inndatafeltene i verktøylinjen for alternativer.
