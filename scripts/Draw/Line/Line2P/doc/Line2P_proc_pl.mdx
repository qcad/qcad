import Close from './Close.png';
import Undo from './Undo.png';

1. Wybierz żądany
   [typ wiersza](/pl/manual/scripts/Draw/Line/doc/Line#type) na pasku narzędzi
   opcji.
2. Określić punkt początkowy pierwszego odcinka linii. Można użyć myszy lub
   wprowadzić współrzędne w konsoli.
3. Określa punkt końcowy odcinka pierwszej linii.
4. Określa punkty końcowe dodatkowych odcinków linii. Kliknąć przycisk
   "Zamknij" na pasku narzędzi opcji, aby zamknąć sekwencję:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Jeśli chcesz cofnąć pojedynczy odcinek linii, możesz to zrobić klikając
   przycisk "Cofnij":
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Długość lub kąt linii można ustawić na stałe, korzystając z odpowiednich opcji na pasku narzędzi opcji.
