import Close from './Close.png';
import Undo from './Undo.png';

1. Выберите нужный
[тип линии](/ru/manual/scripts/Draw/Line/doc/Line) на панели инструментов.
2. Укажите начальную точку первого сегмента линии. Можно использовать мышь или
ввести координаты в консоли.
3. Укажите конечную точку первого сегмента линии.
4. Укажите конечные точки дополнительных сегментов линии. Нажмите кнопку
"Закрыть" на панели инструментов, чтобы закрыть последовательность:

<Image src={Close} alt="Close" width={40} height={40} />  
Если вам нужно отменить один сегмент линии, вы можете сделать это, нажав
кнопку "Отменить":

<Image src={Undo} alt="Undo" width={40} height={40} />
5. Вы можете задать фиксированные значения длины или угла наклона линии с
помощью соответствующих входов на панели инструментов.
