import Close from './Close.png';
import Undo from './Undo.png';

1. Pilih [tipe garis](/id/manual/scripts/Draw/Line/doc/Line#type) yang diinginkan pada bilah alat opsi.
2. Tentukan titik awal segmen garis pertama. Anda dapat menggunakan mouse atau memasukkan koordinat pada konsol.
3. Tentukan titik akhir segmen garis pertama.
4. Tentukan titik akhir segmen garis tambahan. Klik tombol "Close" pada bilah alat opsi untuk menutup rangkaian:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Jika Anda perlu mengurungkan satu segmen garis, Anda dapat melakukannya dengan mengklik tombol "Undo":
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Anda dapat mengatur panjang atau sudut garis ke nilai tetap menggunakan input yang sesuai pada bilah alat opsi.
