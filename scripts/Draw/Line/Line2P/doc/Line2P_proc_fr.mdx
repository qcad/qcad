import Close from './Close.png';
import Undo from './Undo.png';

1. Sélectionnez le
   [type de ligne](/fr/manual/scripts/Draw/Line/doc/Line#type) souhaité dans la barre
   d'outils des options.
2. Spécifiez le point de départ de la première section de ligne. Vous pouvez
   utiliser la souris ou entrer une coordonnée dans la console.
3. Spécifiez le point final du premier segment de ligne.
4. Spécifiez les points d'extrémité des segments de ligne supplémentaires.
   Cliquez sur le bouton "Fermer" dans la barre d'outils des options pour fermer la
   séquence :
   <Image src={Close} alt="Close" width={40} height={40} />  
   Si vous avez besoin d'annuler un seul segment de ligne, vous pouvez le
   faire en cliquant sur le bouton "Annuler"
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Vous pouvez fixer la longueur ou l'angle de la ligne à des valeurs fixes en utilisant les entrées appropriées dans la barre d'outils des options.
