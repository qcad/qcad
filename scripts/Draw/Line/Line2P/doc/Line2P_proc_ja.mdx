import Close from './Close.png';
import Undo from './Undo.png';

1. オプションツールバーで希望の
[線種を](/ja/manual/scripts/Draw/Line/doc/Line)選ぶ。
2. 最初の線分の始点を指定します。マウスを使うか、コンソールに座標を入力する。
3. 最初の線分の終点を指定します。
4. 追加の線分の終点を指定する。オプションツールバーの「閉じる」ボタンをクリックして、シーケンスを閉じる：

<Image src={Close} alt="Close" width={40} height={40} />  
ひとつの線分を元に戻したいときは、"Undo" ボタンをクリックします：

<Image src={Undo} alt="Undo" width={40} height={40} />
5. オプションツールバーの適切な入力を使って、線の長さや角度を固定値に設定することができます。
