import Close from './Close.png';
import Undo from './Undo.png';

1. Escolher o
   [tipo de linha](/pt/manual/scripts/Draw/Line/doc/Line#type) desejada na barra de
   ferramentas de opções.
2. Especificar o ponto de partida do primeiro segmento de linha. Pode utilizar
   o rato ou introduzir uma coordenada na consola.
3. Especificar o ponto final do primeiro segmento de linha.
4. Especificar os pontos finais de segmentos de linha adicionais. Clique no
   botão "Fechar" na barra de ferramentas das opções para fechar a sequência:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Se precisar de desfazer um único segmento de linha, pode fazê-lo clicando
   no botão "Desfazer"
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Você pode definir o comprimento ou o ângulo da linha para valores fixos usando as entradas apropriadas na barra de ferramentas de opções.
