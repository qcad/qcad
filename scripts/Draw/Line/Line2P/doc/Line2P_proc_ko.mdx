import Close from './Close.png';
import Undo from './Undo.png';

1. 옵션 도구 모음에서 원하는 [선 유형](/ko/manual/scripts/Draw/Line/doc/Line#type)을 선택합니다.
2. 첫 번째 선 세그먼트의 시작점을 지정합니다. 마우스를 사용하거나 콘솔에 좌표를 입력할 수 있습니다.
3. 첫 번째 선 세그먼트의 끝점을 지정합니다.
4. 추가 선 세그먼트의 끝점을 지정합니다. 옵션 도구 모음에서 „닫기” 버튼을 클릭하여 시퀀스를 닫습니다:
   <Image src={Close} alt="Close" width={40} height={40} />  
   단일 선 세그먼트를 실행 취소해야 하는 경우 „실행 취소” 버튼을 클릭하여 수행할 수 있습니다:
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. 옵션 도구 모음의 해당 입력란을 사용하여 선의 길이 또는 각도를 고정 값으로 설정할 수 있습니다.
