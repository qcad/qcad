1. Scegliere il
[tipo di linea](/it/manual/scripts/Draw/Line/doc/Line#type) desiderato nella barra
degli strumenti delle opzioni.
2. Specificare il punto di partenza del segmento di prima linea. È possibile
utilizzare il mouse o inserire una coordinata nella console.
3. Specificare l'endpoint del segmento di prima linea.
4. Specificare gli endpoint di altri segmenti di linea. Per chiudere la
sequenza fare clic sul pulsante "Chiudi" nella barra degli strumenti delle
opzioni:

![](Close.png)  
Se è necessario annullare un segmento di linea singolo, è possibile farlo
facendo clic sul pulsante "Annulla":

![](Undo.png)
5. È possibile impostare la lunghezza o l'angolo della linea su valori fissi utilizzando gli appositi input nella barra degli strumenti delle opzioni.
