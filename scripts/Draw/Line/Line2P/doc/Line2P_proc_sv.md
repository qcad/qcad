1. Välj önskad
[linjetyp](../../doc/Line_sv.html) i verktygsfältet för
alternativ.
2. Ange startpunkten för det första linjesegmentet. Du kan använda musen eller
ange en koordinat i konsolen.
3. Ange slutpunkten för det första linjesegmentet.
4. Ange ändpunkterna för ytterligare linjesegment. Klicka på knappen "Close"
(Stäng) i verktygsfältet för alternativ för att stänga sekvensen:

![](Close.png)  
Om du behöver ångra ett enskilt linjesegment kan du göra det genom att
klicka på knappen "Undo" (Ångra)

![](Undo.png)
5. Du kan ställa in linjens längd eller vinkel till fasta värden med hjälp av lämpliga ingångar i verktygsfältet för alternativ.
