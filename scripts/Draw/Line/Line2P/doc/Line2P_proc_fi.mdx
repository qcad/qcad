import Close from './Close.png';
import Undo from './Undo.png';

1. Valitse haluamasi [viivatyyppi](/fi/manual/scripts/Draw/Line/doc/Line#type) asetustyökalurivillä.
2. Määritä ensimmäisen viivasegmentin aloituspiste. Voit käyttää hiirtä tai syöttää koordinaatin konsoliin.
3. Määritä ensimmäisen viivasegmentin päätepiste.
4. Määritä lisäviivasegmenttien päätepisteet. Sulje jono napsauttamalla asetustyökalurivin painiketta "Close":
   <Image src={Close} alt="Close" width={40} height={40} />  
   Jos sinun täytyy kumota yksittäinen viivasegmentti, voit tehdä sen napsauttamalla painiketta "Undo":
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Voit asettaa viivan pituuden tai kulman kiinteisiin arvoihin käyttämällä asetustyökalurivin vastaavia syöttökenttiä.
