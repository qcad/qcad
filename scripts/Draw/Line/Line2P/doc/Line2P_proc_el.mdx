import Close from './Close.png';
import Undo from './Undo.png';

1. Επιλέξτε τον επιθυμητό [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line) στη γραμμή εργαλείων επιλογών.
2. Ορίστε το αρχικό σημείο του πρώτου τμήματος γραμμής. Μπορείτε να χρησιμοποιήσετε το ποντίκι ή να εισαγάγετε μια συντεταγμένη στην κονσόλα.
3. Ορίστε το τελικό σημείο του πρώτου τμήματος γραμμής.
4. Ορίστε τα τελικά σημεία των πρόσθετων τμημάτων γραμμής. Κάντε κλικ στο κουμπί «Κλείσιμο» στη γραμμή εργαλείων επιλογών για να κλείσετε την ακολουθία:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Αν χρειαστεί να αναιρέσετε ένα μεμονωμένο τμήμα γραμμής, μπορείτε να το κάνετε κάνοντας κλικ στο κουμπί «Αναίρεση»:
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Μπορείτε να ορίσετε το μήκος ή τη γωνία της γραμμής σε σταθερές τιμές χρησιμοποιώντας τα κατάλληλα πεδία εισαγωγής στη γραμμή εργαλείων επιλογών.
