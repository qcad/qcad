import Close from './Close.png';
import Undo from './Undo.png';

1. Válassza ki a kívánt [vonaltípust](/hu/manual/scripts/Draw/Line/doc/Line#type) a felső eszköztárban.
2. Adja meg az első vonalszakasz kezdőpontját. Használhatja az egeret, vagy megadhat egy koordinátát a konzolban.
3. Adja meg az első vonalszakasz végpontját.
4. Adja meg a további vonalszakaszok végpontjait. Kattintson a felső eszköztár „Bezárás” gombjára a sorozat lezárásához:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Ha egyetlen vonalszakaszt szeretne visszavonni, ezt a „Visszavonás” gombra kattintva teheti meg:
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. A vonal hosszát vagy szögét rögzített értékekre állíthatja a felső eszköztár megfelelő beviteli mezőivel.
