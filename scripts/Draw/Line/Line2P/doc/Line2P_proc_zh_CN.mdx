import Close from './Close.png';
import Undo from './Undo.png';

1. 在选项工具栏中选择所需的[线型](/zh_CN/manual/scripts/Draw/Line/doc/Line#type)。
2. 指定第一条线段的起点。您可以使用鼠标或在控制台中输入坐标。
3. 指定第一条线段的端点。
4. 指定其他线段的端点。单击选项工具栏中的"Close"按钮以闭合序列：
   <Image src={Close} alt="Close" width={40} height={40} />  
   如果您需要撤销单条线段，可以单击"Undo"按钮：
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. 您可以使用选项工具栏中的相应输入框将直线的长度或角度设置为固定值。
