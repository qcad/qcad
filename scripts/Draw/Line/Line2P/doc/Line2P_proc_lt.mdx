import Close from './Close.png';
import Undo from './Undo.png';

1. Parinkčių įrankių juostoje pasirinkite norimą [linijos tipą](/lt/manual/scripts/Draw/Line/doc/Line#type).
2. Nurodykite pirmojo linijos segmento pradžios tašką. Galite naudoti pelę arba įvesti koordinatę konsolėje.
3. Nurodykite pirmojo linijos segmento galinį tašką.
4. Nurodykite papildomų linijos segmentų galinius taškus. Spustelėkite mygtuką „Close" parinkčių įrankių juostoje, kad uždarytumėte seką:
   <Image src={Close} alt="Close" width={40} height={40} />  
   Jei reikia atšaukti vieną linijos segmentą, tai galite padaryti spustelėdami mygtuką „Undo":
   <Image src={Undo} alt="Undo" width={40} height={40} />
5. Linijos ilgį ar kampą galite nustatyti į fiksuotas reikšmes naudodami atitinkamus įvesties laukus parinkčių įrankių juostoje.
