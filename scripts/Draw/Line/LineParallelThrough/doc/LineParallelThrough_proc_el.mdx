1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Εισαγάγετε τον αριθμό των οντοτήτων που θα δημιουργηθούν στη γραμμή εργαλείων επιλογών.
3. Κάντε κλικ στην οντότητα βάσης.
4. Κάντε κλικ στη θέση από την οποία διέρχεται το πρώτο παράλληλο ή ομόκεντρο τόξο ή
   κύκλος.
