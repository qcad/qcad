1. Scegliere il
[tipo di riga](/it/manual/scripts/Draw/Line/doc/Line#type) desiderato nella barra
degli strumenti delle opzioni.
2. Immettere l'angolo relativo nella barra degli strumenti delle opzioni.
3. Fare clic sull'entità di base esistente.
4. Posizionare la linea con il mouse o inserire una coordinata nella riga di
comando.
