Ezzel az eszközzel meglévő elemekhez viszonyított relatív szögű vonalakat hozhat létre. A
meglévő elem lehet vonal vagy ív / kör. Az ívhez viszonyított 0 fokos relatív szögű
vonalak érintők vagy érintőkkel párhuzamosak. Az ívhez vagy vonalhoz viszonyított 90 fokos
relatív szögű vonalak merőleges vonalak.
