1. Choisissez le
[type de ligne](/fr/manual/scripts/Draw/Line/doc/Line#type) désiré dans la barre
d'outils des options.
2. Saisissez l'angle relatif dans la barre d'outils des options.
3. Cliquez sur l'entité de base existante.
4. Placez la ligne avec la souris ou entrez une coordonnée dans la ligne de
commande.
