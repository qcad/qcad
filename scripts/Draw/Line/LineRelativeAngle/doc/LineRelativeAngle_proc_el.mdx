1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Εισαγάγετε τη σχετική γωνία στη γραμμή εργαλείων επιλογών.
3. Κάντε κλικ στην υπάρχουσα οντότητα βάσης.
4. Τοποθετήστε τη γραμμή με το ποντίκι ή εισαγάγετε μια συντεταγμένη στη γραμμή
   εντολών.
