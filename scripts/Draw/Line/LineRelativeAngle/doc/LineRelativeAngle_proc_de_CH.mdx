1. Wählen Sie den gewünschten
   [Linientyp](/de_CH/manual/scripts/Draw/Line/doc/Line#type) in der
   Optionenwerkzeugleiste.
2. Geben Sie den relativen Winkel in der Optionenwerkzeugleiste ein.
3. Klicken Sie das existierende Basisobjekt.
4. Positionieren Sie die Linie mit der Maus oder geben Sie eine Koordinate in
   die Kommandozeile ein.
