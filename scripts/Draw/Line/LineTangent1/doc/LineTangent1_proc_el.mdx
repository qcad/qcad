1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Χρησιμοποιήστε το ποντίκι για να ορίσετε τη θέση του αρχικού σημείου της γραμμής ή
   εισαγάγετε μια συντεταγμένη στη γραμμή εντολών.
3. Κάντε κλικ στην οντότητα προς την οποία θέλετε να δημιουργήσετε την εφαπτομένη. Συνήθως, δύο
   εφαπτομένες είναι δυνατές. Καθώς μετακινείτε το ποντίκι, μπορείτε να δείτε μια προεπισκόπηση της
   εφαπτομένης που θα δημιουργηθεί.
