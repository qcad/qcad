1. Wählen Sie den gewünschten
[Linientyp](/de/manual/scripts/Draw/Line/doc/Line#type) in der
Optionenwerkzeugleiste.
2. Bestimmen Sie den Startpunkt der Tangente mit der Maus oder geben Sie eine
Koordinate in die Kommandozeile ein.
3. Wählen Sie das Objekt an das Sie die Tangente konstruieren wollen.
Normalerweise sind zwei Tangenten möglich. Wenn Sie die Maus bewegen können Sie
jeweils eine Vorschau derjenigen Tangente sehen, die konstruiert würde, wenn
Sie klicken.
