1. Välj önskad
[linjetyp](/sv/manual/scripts/Draw/Line/doc/Line#type) i verktygsfältet för
alternativ.
2. Använd musen för att ange platsen för linjens startpunkt eller ange en
koordinat på kommandoraden.
3. Klicka på den enhet som du vill skapa tangenten till. Vanligtvis är två
tangenter möjliga. När du flyttar musen kan du se en förhandsvisning av den
tangent som kommer att skapas.
