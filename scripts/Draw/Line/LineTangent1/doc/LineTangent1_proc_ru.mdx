1. Выберите нужный [тип линии](/ru/manual/scripts/Draw/Line/doc/Line#type) на
   панели опций.
2. С помощью мыши укажите положение начальной точки линии или введите
   координату в командной строке.
3. Щёлкните объект, к которому хотите построить касательную. Обычно возможны
   две касательные. При перемещении мыши можно увидеть предпросмотр касательной,
   которая будет создана.
