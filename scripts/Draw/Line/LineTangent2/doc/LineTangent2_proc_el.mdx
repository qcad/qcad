1. Επιλέξτε τον επιθυμητό
   [τύπο γραμμής](/el/manual/scripts/Draw/Line/doc/Line#type) στη γραμμή εργαλείων
   επιλογών.
2. Κάντε κλικ στην πρώτη οντότητα τόξου ή κύκλου προς την οποία θέλετε να δημιουργήσετε την
   εφαπτομένη.
3. Κάντε κλικ στη δεύτερη οντότητα τόξου ή κύκλου προς την οποία θέλετε να δημιουργήσετε την
   εφαπτομένη. Συνήθως, τέσσερις εφαπτομένες είναι δυνατές. Καθώς μετακινείτε το ποντίκι, μπορείτε
   να δείτε μια προεπισκόπηση της εφαπτομένης που θα δημιουργηθεί.
