import LineBisectorFigure from './LineBisectorFigure.png';

1. Выберите нужный
   [тип линии](/ru/manual/scripts/Draw/Line/doc/Line#type) на панели
   инструментов.
2. На панели инструментов введите длину биссектрис(ы), начиная от точки
   пересечения двух линий. Во втором текстовом поле выберите количество угловых
   биссектрис, которые вы хотите создать. По умолчанию — «1», но вы также можете
   создать несколько биссектрис, как показано здесь:
   <Image src={LineBisectorFigure} alt="LineBisectorFigure" width={280} height={100} />
3. Щёлкните первый линейный объект, ограничивающий угол.
4. Щёлкните второй линейный объект, наведя курсор мыши на ту сторону линии, на
   которой вы хотите создать угловую биссектрису(ы). Предпросмотр показывает
   угловую биссектрису(ы) до их создания.
