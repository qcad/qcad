import LineBisectorFigure from './LineBisectorFigure.png';

1. オプションツールバーで希望の
   [線種を](/ja/manual/scripts/Draw/Line/doc/Line#type)選びます。
2. オプションツールバーの中で、2つのラインの交点からスタートして、二等分線の長さを入力します。第2のテキストボックスでは、作成したい角の二等分線の数を選んでください。デフォルトは「1」です。しかし、さらに、ここに示されるような多数の二等分線を作成することができます:
   <Image src={LineBisectorFigure} alt="LineBisectorFigure" width={280} height={100} />
3. 角度を制限する最初のラインエンティティをクリックします。
4. マウス・カーソルで、角の二等分線を作成したい第2のラインエンティティをクリックします。それらが作成される前に、プレビューとして角の二等分線を表示します。
