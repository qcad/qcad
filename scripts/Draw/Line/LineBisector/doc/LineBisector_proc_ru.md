1. In the options tool bar enter the length of the bisector(s), starting from
 the intersection point of the two lines. In the second text box, choose the
 number of angle bisectors you want to create. The default is '1' but you can
 also create multiple bisectors as shown here:  
![](LineBisectorFigure.png)
2. Click the first line entity which limits the angle.
3. Click the second line entity with the mouse cursor on that side of the
 line on which you want to create the angle bisector(s). The preview shows the
 angle bisector(s) before they are created.
