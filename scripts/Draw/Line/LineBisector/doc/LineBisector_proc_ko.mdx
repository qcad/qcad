1. 옵션 도구 모음에서 원하는
   [선 유형](/ko/manual/scripts/Draw/Line/doc/Line#type)을 선택합니다.
2. 옵션 도구 모음에 두 선의 교차점에서 시작하는 이등분선의 길이를
   입력합니다. 두 번째 텍스트 상자에서 만들려는 각 이등분선의 개수를
   선택합니다. 기본값은 „1”이지만 여기에 표시된 것처럼 여러 개의
   이등분선을 만들 수도 있습니다:

<Image src={LineBisectorFigure} alt="LineBisectorFigure" width={280} height={100} />

3. 각을 제한하는 첫 번째 선 개체를 클릭합니다.
4. 각 이등분선을 만들려는 선의 쪽에 마우스 커서를 두고 두 번째 선
   개체를 클릭합니다. 미리 보기는 만들어지기 전의 각 이등분선을
   표시합니다.
