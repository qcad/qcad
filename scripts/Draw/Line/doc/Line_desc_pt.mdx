As linhas são objetos retos definidos por dois pontos. As ferramentas de linha oferecem várias formas de construir uma linha, por exemplo de um ponto inicial a um ponto final, paralelamente a um objeto existente, tangencialmente a círculos ou arcos ou ortogonalmente a outra linha.

*Tipos de linha*

A maioria das ferramentas de linha permite escolher o tipo de linha a criar na
 barra de ferramentas de opções. Os tipos de linha disponíveis são:

- Auto:
  <Image src={TypeAuto} alt="TypeAuto" width={40} height={40} />
  Cria automaticamente uma linha do mesmo tipo que outra linha selecionada. Isto aplica-se à ferramenta de linha paralela.
- Segmento de linha:
  <Image src={TypeSegment} alt="TypeSegment" width={40} height={40} />
  Cria segmentos de linha de um ponto inicial a um ponto final.
- Linha infinita:
  <Image src={TypeXLine} alt="TypeXLine" width={40} height={40} />
  Cria linhas infinitas que passam por dois pontos dados. Por vezes são chamadas linhas X ou linhas de construção.
- Raios:
  <Image src={TypeRay} alt="TypeRay" width={40} height={40} />
  Cria raios a partir de um ponto inicial dado, através de outro ponto com um comprimento infinito.
