直線是由兩個點定義的直實體。直線工具提供了多種建構直線的方法，例如從起點到端點、平行於現有實體、與圓或圓弧相切或與另一條直線正交。

### 線型 {#type}

大多數直線工具允許您在選項工具列中選擇要建立的直線類型。可用的線型有：

- Auto:
  <Image src={TypeAuto} alt="TypeAuto" width={40} height={40} />
  自動建立與所選另一條直線相同類型的直線。這適用於平行線工具。
- Line Segment:
  <Image src={TypeSegment} alt="TypeSegment" width={40} height={40} />
  建立從起點到端點的線段。
- Infinite Line:
  <Image src={TypeXLine} alt="TypeXLine" width={40} height={40} />
  建立通過兩個給定點的無限長直線。這些有時被稱為 X 線或建構線。
- Rays:
  <Image src={TypeRay} alt="TypeRay" width={40} height={40} />
  從給定的起點建立射線，通過另一個點並具有無限長度。
