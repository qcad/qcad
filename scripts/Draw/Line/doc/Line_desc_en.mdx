Lines are straight entities defined by two points. The line tools offer various ways to construct a line, for example from a start point to an end point, parallel to an existing entity, tangentially to circles or arcs or orthogonally to another line.

### Line Types {#type}

Most line tools allow you to choose the type of line to create in the
 options toolbar. The line types available are:

- Auto:
  <Image src={TypeAuto} alt="TypeAuto" width={40} height={40} />
  Automatically create a line of the same type as a chosen other line. This applies for the parallel line tool.
- Line Segment:
  <Image src={TypeSegment} alt="TypeSegment" width={40} height={40} />
  Creates line segments from a start point to an end point.
- Infinite Line:
  <Image src={TypeXLine} alt="TypeXLine" width={40} height={40} />
  Creates infinite lines that go through two given points. These are sometimes called X-lines or construction lines.
- Rays:
  <Image src={TypeRay} alt="TypeRay" width={40} height={40} />
  Creates rays from a given start point, through another point with an infinite length.
