Linien sind gerade Objekte, die durch zwei Punkte definiert sind. Die Linienwerkzeuge bieten verschiedene Möglichkeiten, eine Linie zu konstruieren, zum Beispiel von einem Startpunkt zu einem Endpunkt, parallel zu einem bestehenden Objekt, tangential an Kreise oder Bögen oder orthogonal zu einer anderen Linie.

*Linientypen*

Die meisten Linienwerkzeuge erlauben es, den Typ der zu erstellenden Linie
 in der Optionenwerkzeugleiste zu wählen. Es stehen folgende Linientypen zur
 Verfügung:

- Auto:  
  <Image src={TypeAuto} alt="TypeAuto" width={40} height={40} />  
  Erstellt automatisch eine Linie des gleichen Typs wie eine
  ausgewählte andere Linie. Diese Option ist für das Werkzeug zum Erstellen
  von parallelen Linien verfügbar.
- Liniensegment:  
  <Image src={TypeSegment} alt="TypeSegment" width={40} height={40} />  
  Erstellt Liniensegmente von einem Startpunkt bis zu einem
  Endpunkt.
- Unendliche Linie  
  <Image src={TypeXLine} alt="TypeXLine" width={40} height={40} />  
  Erzeugt unendliche Linien, die durch zwei vorgegebene Punkte
  verlaufen. Diese werden manchmal auch als X-Linien oder Konstruktionslinien
  bezeichnet.
- Strahlen:  
  <Image src={TypeRay} alt="TypeRay" width={40} height={40} />  
  Erzeugt Strahlen von einem bestimmten Startpunkt aus, durch einen
  anderen Punkt mit einer unendlichen Länge.
