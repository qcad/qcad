1. Επιλέξτε τη
   [λειτουργία επιλογής](/el/manual/scripts/Select/doc/Select#mode) στη γραμμή εργαλείων
   επιλογών.
2. Χρησιμοποιήστε το ποντίκι για να επιλέξετε μία οντότητα του περιγράμματος που θέλετε να επιλέξετε. Το
   εργαλείο στη συνέχεια αναζητά από τα τελικά σημεία της επιλεγμένης οντότητας και προς τις δύο κατευθύνσεις
   για οντότητες που είναι άμεσα ή έμμεσα συνδεδεμένες με αυτή την οντότητα. Όλες οι
   συνδεδεμένες οντότητες επιλέγονται ή αποεπιλέγονται ανάλογα με την επιλεγμένη λειτουργία
   επιλογής.

Αντί να χρησιμοποιήσετε αυτό το εργαλείο, μπορείτε απλώς να κάνετε διπλό κλικ σε μια οντότητα για να επιλέξετε
αυτήν και όλες τις συνδεδεμένες οντότητες.
