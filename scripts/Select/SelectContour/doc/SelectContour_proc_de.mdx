1. Wählen Sie den
[Selektionsmodus](/de/manual/scripts/Select/doc/Select#mode) in der
Optionenwerkzeugleiste.
2. Benutzen Sie die Maus um ein Objekt der Kontur, die Sie selektieren wollen,
anzuklicken. Das Werkzeug sucht dann automatisch alle an das Objekt
angeschlossenen Objekte und selektiert oder deselektiert diese, je nach
Selektionsmodus.

Statt dieses Werkzeug zu gebrauchen, können Sie auch einfach auf ein Objekt
doppelklicken, um das Objekt und alle verbundenen Objekte auszuwählen.
