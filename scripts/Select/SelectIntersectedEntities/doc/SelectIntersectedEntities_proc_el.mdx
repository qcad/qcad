1. Επιλέξτε τη
   [λειτουργία επιλογής](/el/manual/scripts/Select/doc/Select#mode) στη γραμμή εργαλείων
   επιλογών.
2. Κάντε κλικ στο αρχικό σημείο της γραμμής τομής.
3. Κάντε κλικ στο τελικό σημείο της γραμμής τομής.
4. Όλες οι τεμνόμενες οντότητες επιλέγονται ή αποεπιλέγονται ανάλογα με
   τη λειτουργία επιλογής.
