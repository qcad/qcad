1. Wählen Sie den
[Selektionsmodus](../../doc/Select_de.html#mode) in der
Optionenwerkzeugleiste.
2. Bestimmen Sie den Startpunkt der Linie mit der Maus oder indem Sie eine
Koordinate eingeben.
3. Bestimmen Sie den Endpunkt der Linie.
4. Alle Objekte, die von der gezeichneten Linie geschnitten werden selektiert
oder deselektiert, je nach Selektionsmodus.
