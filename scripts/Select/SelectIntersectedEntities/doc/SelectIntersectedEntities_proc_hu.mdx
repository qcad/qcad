1. Válassza ki a
   [kijelölési módot](/hu/manual/scripts/Select/doc/Select#mode) a felső
   eszköztárban.
2. Kattintson a metszővonal kezdőpontjára.
3. Kattintson a metszővonal végpontjára.
4. Az összes metszett elem kijelölésre vagy kijelölésének megszüntetésére kerül a
   kijelölési módtól függően.
