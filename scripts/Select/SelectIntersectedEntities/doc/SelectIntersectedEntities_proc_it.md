1. Selezionare la
[modalità di selezione](../../doc/Select_it.html#mode) nella barra
degli strumenti delle opzioni.
2. Fare clic sul punto di partenza della linea di intersezione.
3. Fare clic sul punto finale della linea di intersezione.
4. Tutte le entità intersecate corrispondono e vengono selezionate o
deselezionate a seconda della modalità di selezione.
