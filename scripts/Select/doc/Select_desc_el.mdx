### Διασταυρούμενη επιλογή

Ορισμένα εργαλεία επιλογής μπορούν να χρησιμοποιηθούν σε λειτουργία διασταυρούμενης επιλογής. Σε αυτή τη λειτουργία,
 επιλέγονται όχι μόνο οι οντότητες που βρίσκονται εντελώς εντός μιας δοθείσας περιοχής αλλά
 και οι οντότητες που βρίσκονται μόνο εν μέρει εντός της περιοχής. Αυτή η επιλογή είναι επίσης
 γνωστή ως «διασταυρούμενη επιλογή».

### Λειτουργία επιλογής

Ορισμένα εργαλεία επιλογής σάς επιτρέπουν να επιλέξετε μια λειτουργία επιλογής στη γραμμή εργαλείων
 επιλογών. Οι διαθέσιμες λειτουργίες επιλογής είναι:

- Αντικατάσταση επιλογής:  
  <Image src={SelectionModeReplace} alt="SelectionModeReplace" width={40} height={40} />  
  Αντικαθιστά την τρέχουσα επιλογή με τη νέα επιλογή που γίνεται από το
  ενεργό εργαλείο επιλογής (προεπιλογή).
- Προσθήκη στην επιλογή:  
  <Image src={SelectionModeAdd} alt="SelectionModeAdd" width={40} height={40} />  
  Προσθέτει την επιλογή στην τρέχουσα επιλογή.
- Αφαίρεση από την επιλογή:  
  <Image src={SelectionModeRemove} alt="SelectionModeRemove" width={40} height={40} />  
  Αφαιρεί (αφαιρεί) την επιλογή από την τρέχουσα επιλογή.
- Τομή:  
  <Image src={SelectionModeIntersect} alt="SelectionModeIntersect" width={40} height={40} />  
  Επιλέγονται μόνο οι οντότητες που ήταν ήδη επιλεγμένες και που πληρούν τα κριτήρια
  του εργαλείου επιλογής.
