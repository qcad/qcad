*Korsval*

Vissa markeringsverktyg kan användas i korsmarkeringsläge. I det här läget
 väljs inte bara enheter som ligger helt och hållet inom ett visst område utan
 även enheter som bara delvis ligger inom området. Detta urval kallas också
 "korsurval".

*Urvalsläge*

Med vissa urvalsverktyg kan du välja urvalsmetod i verktygsfältet för
 alternativ. De urvalslägen som är tillgängliga är följande:

- Ersätt urval:  
  <Image src={SelectionModeReplace} alt="SelectionModeReplace" width={40} height={40} />  
Ersätter det aktuella urvalet med det nya urvalet som görs av det
 aktiva urvalsverktyget (standard).
- Lägg till urval:  
  <Image src={SelectionModeAdd} alt="SelectionModeAdd" width={40} height={40} />  
Lägger till urvalet till det aktuella urvalet.
- Ta bort från urvalet:  
  <Image src={SelectionModeRemove} alt="SelectionModeRemove" width={40} height={40} />  
Tar bort (subtraherar) urvalet från det aktuella urvalet.
- Intersect:  
  <Image src={SelectionModeIntersect} alt="SelectionModeIntersect" width={40} height={40} />  
Endast enheter som redan var markerade och som matchar
 urvalsverktygets kriterier markeras.
