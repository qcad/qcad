Korsval

Vissa markeringsverktyg kan användas i korsmarkeringsläge. I det här läget väljs inte bara enheter som ligger helt och hållet inom ett visst område utan även enheter som bara delvis ligger inom området. Detta urval kallas också "korsurval".

Urvalsläge

Med vissa urvalsverktyg kan du välja urvalsmetod i verktygsfältet för alternativ. De urvalslägen som är tillgängliga är följande:

  • Ersätt urval:

    Ersätter det aktuella urvalet med det nya urvalet som görs av det aktiva urvalsverktyget (standard).
  • Lägg till urval:

    Lägger till urvalet till det aktuella urvalet.
  • Ta bort från urvalet:

    Tar bort (subtraherar) urvalet från det aktuella urvalet.
  • Intersect:

    Endast enheter som redan var markerade och som matchar urvalsverktygets kriterier markeras.
