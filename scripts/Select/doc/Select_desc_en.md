

*Cross Selection*

Some selection tools can be used in cross selection mode. In this mode,
 not only entities that are completely inside a given area are selected but
 also entities that are only partly inside the area. This selection is also
 known as "cross selection".

*Selection Mode*

Some selection tools allow you to choose a selection mode in the options
 toolbar. The selection modes that are available are:

- Replace selection:  
![](SelectionModeReplace.png)  
Replaces the current selection with the new selection made by the
 active selection tool (default).
- Add to selection:  
![](SelectionModeAdd.png)  
Adds the selection to the current selection.
- Remove from selection:  
![](SelectionModeRemove.png)  
Removes (subtracts) the selection from the current selection.
- Intersect:  
![](SelectionModeIntersect.png)  
Only entities that were already selected and that match the criteria
 of the selection tool are selected.
