1. Välj
   [urvalsläge](/sv/manual/scripts/Select/doc/Select#mode) i verktygsfältet för
   alternativ.
2. Markera kryssrutan för
   [korsurval](/sv/manual/scripts/Select/doc/Select#cross) för att även välja
   enheter som skär den givna rektangeln.
3. Använd musen för att rita upp urvalsrektangeln. Alla matchande enheter
   markeras eller avmarkeras beroende på urvalsläget.
