1. Seleccione el
[modo de selección](/es/manual/scripts/Select/doc/Select#mode) en la barra de
herramientas de opciones.
2. Marque la casilla de verificación para
[selección cruzada](/es/manual/scripts/Select/doc/Select#cross) para seleccionar
también las entidades que intersectan el rectángulo dado.
3. Usa el ratón para dibujar el rectángulo de selección. Todas las entidades
coincidentes se seleccionan o deseleccionan dependiendo del modo de
selección.
