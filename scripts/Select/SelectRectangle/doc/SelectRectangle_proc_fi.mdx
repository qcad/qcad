1. Valitse asetustyökalurivillä [valintatila](/fi/manual/scripts/Select/doc/Select#mode).
2. Valitse [ristivalinnan](/fi/manual/scripts/Select/doc/Select#cross) valintaruutu valitaksesi myös kohteet, jotka leikkaavat annetun suorakulmion.
3. Piirrä hiirellä valintasuorakulmio. Kaikki sopivat kohteet valitaan tai niiden valinta poistetaan valintatilan mukaan.
