1. Sélectionnez le
[mode de sélection](../../doc/Select_fr.html#mode) dans la barre
d'outils des options.
2. Cochez la case
[sélection croisée](../../doc/Select_fr.html#cross) pour
sélectionner également les entités qui croisent le rectangle donné.
3. Utilisez la souris pour dessiner le rectangle de sélection. Toutes les
entités correspondantes sont sélectionnées ou désélectionnées en fonction du
mode de sélection.
