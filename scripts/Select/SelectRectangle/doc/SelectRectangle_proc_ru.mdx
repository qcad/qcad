1. Выберите [режим выделения](/ru/manual/scripts/Select/doc/Select#mode) на
   панели опций.
2. Установите флажок [перекрёстного
   выделения](/ru/manual/scripts/Select/doc/Select#cross), чтобы также выделять
   объекты, пересекающие заданный прямоугольник.
3. С помощью мыши нарисуйте прямоугольник выделения. Все подходящие объекты
   выделяются или с них снимается выделение в зависимости от режима выделения.
