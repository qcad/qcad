1. Odaberite
   [način odabira](/hr/manual/scripts/Select/doc/Select#mode) u alatnoj traci
   opcija.
2. Označite potvrdni okvir za
   [križni odabir](/hr/manual/scripts/Select/doc/Select#cross) kako biste odabrali i
   entitete koji sijeku zadani pravokutnik.
3. Pomoću miša nacrtajte pravokutnik odabira. Svi odgovarajući entiteti
   odabiru se ili im se poništava odabir ovisno o načinu odabira.
