1. Escolher o
   [modo de selecção](/pt/manual/scripts/Select/doc/Select#mode) na barra de
   ferramentas de opções.
2. Assinalar a caixa de
   [selecção cruzada](/pt/manual/scripts/Select/doc/Select#cross) para seleccionar
   também as entidades que intersectam o rectângulo dado.
3. Utilizar o rato para desenhar o rectângulo de selecção. Todas as entidades
   correspondentes são seleccionadas ou desmarcadas, dependendo do modo de
   selecção.
