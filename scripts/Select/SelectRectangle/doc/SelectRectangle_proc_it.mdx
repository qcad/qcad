1. Selezionare la
   [modalità di selezione](/it/manual/scripts/Select/doc/Select#mode) nella barra
   degli strumenti delle opzioni.
2. Spuntare la casella di controllo per la
   [selezione incrociata](/it/manual/scripts/Select/doc/Select#cross) per
   selezionare anche le entità che intersecano il rettangolo dato.
3. Utilizzare il mouse per disegnare il rettangolo di selezione. Tutte le
   entità corrispondenti vengono selezionate o deselezionate a seconda della
   modalità di selezione.
