1. Opciju rīkjoslā izvēlieties [atlases režīmu](/lv/manual/scripts/Select/doc/Select#mode).
2. Atzīmējiet izvēles rūtiņu [krusteniskajai atlasei](/lv/manual/scripts/Select/doc/Select#cross), lai atlasītu arī vienības, kas krusto norādīto taisnstūri.
3. Izmantojiet peli, lai zīmētu atlases taisnstūri. Visas atbilstošās vienības tiek atlasītas vai atlase noņemta atkarībā no atlases režīma.
