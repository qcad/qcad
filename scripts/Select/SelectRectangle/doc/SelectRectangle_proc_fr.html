1. Sélectionnez le mode de sélection dans la barre d'outils des options.
  2. Cochez la case sélection croisée pour sélectionner également les entités qui croisent le rectangle donné.
  3. Utilisez la souris pour dessiner le rectangle de sélection. Toutes les entités correspondantes sont sélectionnées ou désélectionnées en fonction du mode de sélection.
