1. Виберіть [режим вибору](/uk/manual/scripts/Select/doc/Select#mode) на панелі параметрів.
2. Установіть прапорець для [перехресного вибору](/uk/manual/scripts/Select/doc/Select#cross), щоб також вибрати об'єкти, які перетинають заданий прямокутник.
3. Використовуйте мишу, щоб намалювати прямокутник вибору. Усі відповідні об'єкти вибираються або скасовуються залежно від режиму вибору.
