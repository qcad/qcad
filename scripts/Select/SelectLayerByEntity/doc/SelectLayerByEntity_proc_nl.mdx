1. Kies in de opties werkbalk de
   [selectiemodus](/nl/manual/scripts/Select/doc/Select#mode).
2. Gebruik de muis om één object te kiezen op de laag die u wilt
   (de-)selecteren. Alle objecten die zich op dezelfde laag bevinden als het
   object die u hebt geselecteerd, worden geselecteerd of gedeselecteerd,
   afhankelijk van de selectiemodus.
