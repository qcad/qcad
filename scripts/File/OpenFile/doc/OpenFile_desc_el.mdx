Χρησιμοποιήστε αυτή την εντολή για να ανοίξετε ένα αρχείο σχεδίου. Το αρχείο φορτώνεται και εμφανίζεται σε
ένα νέο παράθυρο εντός του παραθύρου της εφαρμογής QCAD. Μπορείτε να εναλλάσσεστε μεταξύ
διαφορετικών φορτωμένων σχεδίων χρησιμοποιώντας το μενού «Παράθυρα» ή τις καρτέλες στο επάνω μέρος της
περιοχής σχεδίασης.
