 Δημιουργήστε νέα, κενά σχέδια χρησιμοποιώντας αυτή την εντολή. Τα νέα σχέδια είναι εντελώς
κενά (καμία οντότητα, μόνο ένα προεπιλεγμένο στρώμα, κανένα μπλοκ). Ως συχνός χρήστης του
QCAD, ίσως θέλετε αντ' αυτού να δημιουργήσετε πρότυπα σχεδίων με τα στρώματα και τα
μπλοκ που χρησιμοποιείτε περισσότερο και πιθανώς ένα περίγραμμα σχεδίου. Μόλις έχετε ένα τέτοιο
πρότυπο μπορείτε να το φορτώσετε αντί να ξεκινάτε ένα νέο σχέδιο από την αρχή.  
Ανατρέξτε επίσης στην εντολή [Νέο από πρότυπο](/el/manual/scripts/Pro/File/NewFromTemplate/doc/NewFromTemplate) για να μάθετε πώς να χρησιμοποιείτε τα πρότυπα σχεδίων.
