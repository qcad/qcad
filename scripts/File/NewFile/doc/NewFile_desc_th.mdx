สร้างแบบวาดใหม่ที่ว่างเปล่าโดยใช้คำสั่งนี้ แบบวาดใหม่จะว่างเปล่าอย่างสมบูรณ์ (ไม่มีเอนทิตี มีเพียงเลเยอร์เริ่มต้นหนึ่งเลเยอร์ ไม่มีบล็อก) ในฐานะผู้ใช้ QCAD บ่อยครั้ง คุณอาจต้องการสร้างเทมเพลตแบบวาดที่มีเลเยอร์และบล็อกที่คุณใช้มากที่สุดและอาจมีกรอบแบบวาดแทน เมื่อคุณมีเทมเพลตดังกล่าว คุณสามารถโหลดมันแทนการเริ่มแบบวาดใหม่จากศูนย์
