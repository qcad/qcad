Cree nuevos dibujos vacíos utilizando este comando. Los nuevos dibujos están
completamente vacíos (sin entidades, sólo una capa por defecto, sin bloques).
Como usuario frecuente de QCAD, es posible que desee crear plantillas de dibujo
con las capas y los bloques que más utiliza y probablemente un borde de dibujo.
Una vez que tenga una plantilla de este tipo, puede cargarla en lugar de
empezar un nuevo dibujo desde cero.  
Por favor refiérase también al comando [Nuevo de Plantilla](/es/manual/scripts/Pro/File/NewFromTemplate/doc/NewFromTemplate) para aprender a usar plantillas de dibujo.
