 Mit diesem Befehl können Sie neue Zeichnungen erstellen. Neue Zeichnungen
enthalten keine Objekte, Layer oder Blöcke. Falls Sie häufig mit QCAD arbeiten
können Sie Zeichnungsvorlagen erstellen, die bereits Rahmen, Layer-Definitionen
oder Blöcke enthalten. Um eine neue Zeichnung zu beginnen laden Sie dann
stattdessen eine solche Vorlage anstatt von einer leeren Zeichnung zu starten.  
Bitte beachten Sie dazu die Dokumentation zum Werkzeug [Neu von Template](../../../Pro/File/NewFromTemplate/doc/NewFromTemplate_de.html).
