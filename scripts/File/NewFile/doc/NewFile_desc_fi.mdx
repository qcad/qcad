Luo uusia, tyhjiä piirustuksia tällä komennolla. Uudet piirustukset ovat täysin tyhjiä (ei kohteita, vain yksi oletuskerros, ei lohkoja). QCADin usein käyttäjänä saatat sen sijaan haluta luoda piirustusmalleja, joissa on eniten käyttämäsi kerrokset ja lohkot ja todennäköisesti piirustuksen reunus. Kun sinulla on tällainen malli, voit ladata sen sen sijaan, että aloittaisit uuden piirustuksen alusta.  
Katso myös komento [New from Template](/fi/manual/scripts/Pro/File/NewFromTemplate/doc/NewFromTemplate) oppiaksesi käyttämään piirustusmalleja.
