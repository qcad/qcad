Buat gambar baru yang kosong menggunakan perintah ini. Gambar baru sepenuhnya kosong (tidak ada entitas, hanya satu lapisan default, tidak ada blok). Sebagai pengguna QCAD yang sering, Anda mungkin lebih ingin membuat templat gambar dengan lapisan dan blok yang paling sering Anda gunakan dan mungkin bingkai gambar. Setelah Anda memiliki templat seperti itu, Anda dapat memuatnya alih-alih memulai gambar baru dari awal.  
Silakan lihat juga perintah [New from Template](/id/manual/scripts/Pro/File/NewFromTemplate/doc/NewFromTemplate) untuk mempelajari cara menggunakan templat gambar.
