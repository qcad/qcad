Utilizați această comandă pentru a imprima un desen. Asigurați-vă că comutați întotdeauna la previzualizarea imprimării înainte de imprimare pentru a verifica dacă scara și poziția desenului sunt configurate așa cum doriți.  
O fereastră de dialog de configurare este afișată înainte de a imprima ceva. Utilizați această fereastră de dialog pentru a alege imprimanta și a ajusta setările specifice imprimantei.
