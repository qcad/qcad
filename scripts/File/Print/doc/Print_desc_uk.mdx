Використовуйте цю команду для друку креслення. Обов'язково завжди перемикайтеся на попередній перегляд друку перед друком, щоб перевірити, чи налаштовано масштаб і положення креслення так, як потрібно.  
Перед друком чого-небудь відображається діалогове вікно налаштування. Використовуйте це діалогове вікно для вибору принтера та налаштування специфічних для принтера параметрів.
