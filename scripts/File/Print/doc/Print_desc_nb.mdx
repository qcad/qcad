Bruk denne kommandoen til å skrive ut en tegning. Sørg for alltid å bytte til
forhåndsvisning av utskrift før utskrift for å kontrollere om tegningsskalaen og
-posisjonen er satt opp som ønsket.  
En oppsettsdialog vises før noe skrives ut. Bruk denne dialogen til å
