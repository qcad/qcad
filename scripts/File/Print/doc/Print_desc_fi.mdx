Käytä tätä komentoa piirustuksen tulostamiseen. Varmista, että vaihdat aina tulostuksen esikatseluun ennen tulostamista tarkistaaksesi, että piirustuksen mittakaava ja sijainti on asetettu halutulla tavalla.  
Ennen kuin mitään tulostetaan, näytetään asetusvalintaikkuna. Käytä tätä valintaikkunaa tulostimen valitsemiseen ja tulostinkohtaisten asetusten säätämiseen.
