Χρησιμοποιήστε αυτή την εντολή για να εκτυπώσετε ένα σχέδιο. Φροντίστε να μεταβαίνετε πάντα στην προεπισκόπηση
εκτύπωσης πριν εκτυπώσετε για να ελέγξετε αν η κλίμακα και η θέση του σχεδίου έχουν ρυθμιστεί όπως
επιθυμείτε.  
Ένας διάλογος ρύθμισης εμφανίζεται πριν εκτυπωθεί οτιδήποτε. Χρησιμοποιήστε αυτόν τον διάλογο για να
επιλέξετε τον εκτυπωτή και να προσαρμόσετε τις ειδικές ρυθμίσεις του εκτυπωτή.
