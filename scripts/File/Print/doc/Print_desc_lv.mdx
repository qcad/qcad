Izmantojiet šo komandu, lai drukātu rasējumu. Pārliecinieties, ka vienmēr pārslēdzaties uz drukas priekšskatījumu pirms drukāšanas, lai pārbaudītu, vai rasējuma mērogs un pozīcija ir iestatīti kā vēlaties.  
Pirms kaut kas tiek drukāts, tiek parādīts iestatīšanas dialoglodziņš. Izmantojiet šo dialoglodziņu, lai izvēlētos printeri un pielāgotu printerim specifiskus iestatījumus.
