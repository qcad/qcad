Šia komanda spausdinkite brėžinį. Prieš spausdindami visada persijunkite į spausdinimo peržiūrą, kad patikrintumėte, ar brėžinio mastelis ir padėtis nustatyti kaip pageidaujama.  
Prieš ką nors spausdinant rodomas sąrankos dialogo langas. Šiuo dialogo langu pasirinkite spausdintuvą ir pakoreguokite spausdintuvui būdingus nustatymus.
