Используйте эту команду для печати чертежа. Обязательно всегда переключайтесь
на предпросмотр печати перед печатью, чтобы проверить, правильно ли настроены
масштаб и положение чертежа.  
Перед началом печати отображается диалог настройки. Используйте этот диалог для
выбора принтера и настройки специфичных для принтера параметров.
