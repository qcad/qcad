คำสั่งนำเข้าจะแทรกแบบวาดจากไฟล์บนดิสก์ลงในแบบวาดปัจจุบัน  
เลเยอร์ทั้งหมดที่ใช้โดยแบบวาดที่นำเข้าจะถูกเพิ่มเข้าไปในรายการเลเยอร์ของแบบวาดปัจจุบัน เลเยอร์ที่มีอยู่ที่มีชื่อเดียวกันสามารถเขียนทับได้หากต้องการ (ตัวเลือก „overwrite layers" ในแถบเครื่องมือตัวเลือก)  
การอ้างอิงบล็อกของแบบวาดที่นำเข้าจะถูกแทรกพร้อมกับคำจำกัดความบล็อกที่อ้างอิงถึง บล็อกในแบบวาดปัจจุบันสามารถเขียนทับได้หากต้องการ (ตัวเลือก „overwrite blocks" ในแถบเครื่องมือตัวเลือก)  
แถบเครื่องมือตัวเลือกยังมีเครื่องมือบางอย่างสำหรับปรับสเกล หมุน หรือพลิกแบบวาดที่นำเข้าขณะวางตำแหน่ง
