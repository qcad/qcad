Tuontikomento lisää levyllä olevasta tiedostosta piirustuksen nykyiseen piirustukseen.  
Kaikki tuodun piirustuksen käyttämät kerrokset lisätään nykyisen piirustuksen kerrosluetteloon. Samannimiset olemassa olevat kerrokset voidaan halutessa korvata (asetustyökalurivin vaihtoehto "overwrite layers").  
Tuodun piirustuksen lohkoviitteet lisätään yhdessä niiden lohkomääritysten kanssa, joihin ne viittaavat. Nykyisen piirustuksen lohkot voidaan halutessa korvata (asetustyökalurivin vaihtoehto "overwrite blocks").  
Asetustyökalurivi tarjoaa myös työkaluja tuodun piirustuksen skaalaamiseen, kiertämiseen tai peilaamiseen sen sijoittamisen aikana.
