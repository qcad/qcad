Команда импорта вставляет чертёж из файла на диске в текущий чертёж.  
Все слои, используемые импортированным чертежом, добавляются в список слоёв текущего чертежа. Существующие слои с тем же именем могут быть перезаписаны при необходимости (параметр «перезаписать слои» на панели параметров).  
Ссылки на блоки импортированного чертежа будут вставлены вместе с определениями блоков, на которые они ссылаются. Блоки в текущем чертеже могут быть перезаписаны при необходимости (параметр «перезаписать блоки» на панели параметров).  
Панель параметров также предлагает несколько инструментов для масштабирования, поворота или отражения импортированного чертежа при его размещении.
