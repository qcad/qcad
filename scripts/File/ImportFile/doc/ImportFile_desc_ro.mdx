Comanda de import inserează un desen dintr-un fișier de pe disc în desenul curent.  
Toate straturile utilizate de desenul importat sunt adăugate la lista de straturi a desenului curent. Straturile existente cu același nume pot fi suprascrise dacă se dorește (opțiunea „overwrite layers" din bara de instrumente opțiuni).  
Referințele de bloc ale desenului importat vor fi inserate împreună cu definițiile de bloc la care se referă. Blocurile din desenul curent pot fi suprascrise dacă se dorește (opțiunea „overwrite blocks" din bara de instrumente opțiuni).  
Bara de instrumente opțiuni oferă, de asemenea, câteva instrumente pentru a scala, roti sau răsturna desenul importat în timp ce îl poziționați.
