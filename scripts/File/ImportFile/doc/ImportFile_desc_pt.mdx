O comando de importação insere um desenho de um ficheiro no disco no desenho atual.  
Todas as camadas utilizadas pelo desenho importado são adicionadas à lista de camadas do desenho atual. As camadas existentes com o mesmo nome podem ser substituídas, se pretendido (opção "substituir camadas" na barra de opções).  
As referências de bloco do desenho importado serão inseridas juntamente com as definições de bloco a que se referem. Os blocos no desenho atual podem ser substituídos, se pretendido (opção "substituir blocos" na barra de opções).  
A barra de opções também oferece algumas ferramentas para escalar, rodar ou inverter o desenho importado enquanto o posiciona.
