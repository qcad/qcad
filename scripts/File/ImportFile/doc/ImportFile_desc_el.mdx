Η εντολή εισαγωγής εισάγει ένα σχέδιο από ένα αρχείο στον δίσκο μέσα στο τρέχον
σχέδιο.  
Όλα τα στρώματα που χρησιμοποιούνται από το εισαγόμενο σχέδιο προστίθενται στη
λίστα στρωμάτων του τρέχοντος σχεδίου. Τα υπάρχοντα στρώματα με το ίδιο όνομα μπορούν να
αντικατασταθούν αν θέλετε (επιλογή «αντικατάσταση στρωμάτων» στη γραμμή εργαλείων επιλογών).  
Οι αναφορές μπλοκ του εισαγόμενου σχεδίου εισάγονται μαζί με
τους ορισμούς μπλοκ στους οποίους αναφέρονται. Τα μπλοκ στο τρέχον σχέδιο μπορούν να
αντικατασταθούν αν θέλετε (επιλογή «αντικατάσταση μπλοκ» στη γραμμή εργαλείων επιλογών).  
Η γραμμή εργαλείων επιλογών προσφέρει επίσης ορισμένα εργαλεία για κλιμάκωση, περιστροφή ή αναστροφή του
εισαγόμενου σχεδίου κατά την τοποθέτησή του.
