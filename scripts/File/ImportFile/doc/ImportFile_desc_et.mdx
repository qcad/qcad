Impordikäsk lisab kettal olevast failist joonise praegusesse joonisesse.  
Kõik imporditud joonise kasutatavad kihid lisatakse praeguse joonise kihtide loendisse. Soovi korral saab sama nimega olemasolevad kihid üle kirjutada (valikute tööriistariba suvand "overwrite layers").  
Imporditud joonise ploki viited lisatakse koos plokidefinitsioonidega, millele need viitavad. Soovi korral saab praeguse joonise plokid üle kirjutada (valikute tööriistariba suvand "overwrite blocks").  
Valikute tööriistariba pakub ka tööriistu imporditud joonise skaleerimiseks, pööramiseks või peegeldamiseks selle paigutamise ajal.
