Naredba uvoza umeće crtež iz datoteke na disku u trenutačni
crtež.  
Svi slojevi koje upotrebljava uvezeni crtež dodaju se na popis
slojeva trenutačnog crteža. Postojeći slojevi istog naziva mogu se
prepisati ako se želi („prepiši slojeve” opcija u alatnoj traci opcija).  
Reference bloka uvezenog crteža umeću se zajedno s
definicijama bloka na koje upućuju. Blokovi u trenutačnom crtežu mogu se
prepisati ako se želi („prepiši blokove” opcija u alatnoj traci opcija).  
Alatna traka opcija također nudi neke alate za promjenu mjerila, rotaciju ili zrcaljenje
uvezenog crteža tijekom pozicioniranja.
