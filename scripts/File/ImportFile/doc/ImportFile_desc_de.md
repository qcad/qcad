Dieses Kommando fügt eine Zeichnung von einer Datei in die aktuelle
Zeichnung ein.  
Alle Layer der importierten Zeichnung werden ebenfalls eingefügt.
Existierende Layer können überschrieben werden falls erwünscht (Option "Layer
überschreiben" in der Optionenwerkzeugleiste).  
Blockreferenzen der importierten Zeichnung werden zusammen mit der
dazugehörigen Blockdefinition eingefügt. Bestehende Blöcke können überschrieben
werden wenn erwünscht (Option "Blöcke überschreiben" in der
Optionenwerkzeugleiste).  
Die Optionenwerkzeugleiste offeriert zusätzlich Werkzeuge zum Skalieren,
Rotieren oder Spiegeln der eingefügten Zeichnung.
