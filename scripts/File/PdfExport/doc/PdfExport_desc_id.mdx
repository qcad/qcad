Alat ini mengekspor gambar saat ini ke file PDF. Mengekspor ke PDF sangat mirip dengan mencetak gambar. Anda kemungkinan besar harus menyesuaikan skala gambar dan posisi gambar serta pengaturan halaman sebelum mengekspor gambar ke PDF.  
Untuk alasan ini, QCAD secara otomatis beralih ke pratinjau cetak saat alat ini digunakan. Jika pratinjau cetak tidak sesuai dengan harapan Anda, batalkan ekspor, sesuaikan pengaturan halaman, dan mulai ekspor PDF lagi.
