เครื่องมือนี้ส่งออกแบบวาดปัจจุบันเป็นไฟล์ PDF การส่งออกเป็น PDF คล้ายกับการพิมพ์แบบวาดมาก คุณอาจต้องปรับสเกลแบบวาดและตำแหน่งแบบวาด รวมถึงการตั้งค่าหน้ากระดาษก่อนส่งออกแบบวาดเป็น PDF  
ด้วยเหตุนี้ QCAD จึงสลับไปยังตัวอย่างก่อนพิมพ์โดยอัตโนมัติเมื่อใช้เครื่องมือนี้ หากตัวอย่างก่อนพิมพ์ไม่ตรงกับความคาดหวังของคุณ ให้ยกเลิกการส่งออก ปรับการตั้งค่าหน้ากระดาษ และเริ่มการส่งออก PDF อีกครั้ง
