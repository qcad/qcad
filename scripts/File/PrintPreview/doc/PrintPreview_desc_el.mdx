Η προεπισκόπηση εκτύπωσης δείχνει πώς θα φαίνεται το σχέδιό σας στο εκτυπωμένο αντίγραφο. Χρησιμοποιήστε
το σύνθετο πλαίσιο στη γραμμή εργαλείων επιλογών για να προσαρμόσετε τον συντελεστή κλίμακας του
σχεδίου σας. Αυτή η επιλογή δεν κλιμακώνει ούτε τροποποιεί με άλλον τρόπο τις οντότητες
του σχεδίου σας, ορίζει μόνο τον συντελεστή κλίμακας που εφαρμόζεται για την προσαρμογή του
σχεδίου σας στο χαρτί.  
Τα κουμπιά δεξιά από το σύνθετο πλαίσιο κλίμακας μπορούν να χρησιμοποιηθούν για την αυτόματη προσαρμογή
του σχεδίου σας στο επιλεγμένο χαρτί ή για την αυτόματη προσθήκη σελίδων, ώστε ολόκληρο το
σχέδιό σας να μπορεί να εκτυπωθεί σε πολλές σελίδες.  
Αφού επιλέξετε το εργαλείο «Μετακίνηση θέσης χαρτιού», μπορείτε να μετακινήσετε το χαρτί
ελεύθερα σύροντάς το με το αριστερό κουμπί του ποντικιού. Κάντε κλικ στο δεξί κουμπί του
ποντικιού ή πατήστε το πλήκτρο Escape για να τερματίσετε το εργαλείο.  
Η γραμμή εργαλείων επιλογών προσφέρει ορισμένα άλλα εργαλεία για τη γρήγορη αλλαγή ορισμένων
κοινών προτιμήσεων σχεδίου που είναι επίσης προσβάσιμες στον διάλογο προτιμήσεων σχεδίου.
