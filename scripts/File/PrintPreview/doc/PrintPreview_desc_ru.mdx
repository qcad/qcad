Предпросмотр печати показывает, как ваш чертёж будет выглядеть при печати.
Используйте раскрывающийся список на панели опций для настройки масштабного
коэффициента вашего чертежа. Эта опция фактически не масштабирует и не изменяет
иным образом объекты вашего чертежа, она только задаёт масштабный коэффициент,
применяемый для размещения чертежа на бумаге.  
Кнопки справа от раскрывающегося списка масштаба можно использовать для
автоматического размещения чертежа на выбранной бумаге или для автоматического
добавления страниц, чтобы весь чертёж можно было напечатать на нескольких
страницах.  
После выбора инструмента «Переместить положение бумаги» вы можете свободно
перемещать бумагу, перетаскивая её левой кнопкой мыши. Щёлкните правой кнопкой
мыши или нажмите клавишу Escape, чтобы завершить инструмент.  
Панель опций предлагает некоторые другие инструменты для быстрого изменения
некоторых общих настроек чертежа, которые также доступны в диалоге настроек
чертежа.
