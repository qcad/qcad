Tulostuksen esikatselu näyttää, miltä piirustuksesi näyttää tulosteessa. Käytä asetustyökalurivin yhdistelmäruutua säätääksesi piirustuksesi mittakaavakerrointa. Tämä vaihtoehto ei todellisuudessa skaalaa tai muuten muokkaa piirustuksesi kohteita, se vain asettaa mittakaavakertoimen, jota käytetään piirustuksesi sovittamiseen paperille.  
Mittakaavan yhdistelmäruudun oikealla puolella olevia painikkeita voidaan käyttää piirustuksesi automaattiseen sovittamiseen valitulle paperille tai sivujen automaattiseen lisäämiseen, jotta koko piirustuksesi voidaan tulostaa useille sivuille.  
Kun olet valinnut työkalun "Move Paper Position", voit siirtää paperia vapaasti vetämällä sitä hiiren vasemmalla painikkeella. Napsauta hiiren oikeaa painiketta tai paina Escape-näppäintä lopettaaksesi työkalun.  
Asetustyökalurivi tarjoaa muutamia muita työkaluja joidenkin yleisten piirustuksen asetusten nopeaan muuttamiseen, jotka ovat käytettävissä myös piirustuksen asetusten valintaikkunassa.
