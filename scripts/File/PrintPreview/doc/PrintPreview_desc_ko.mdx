인쇄 미리 보기는 도면이 인쇄물에서 어떻게 보일지 표시합니다. 옵션 도구
모음의 콤보 상자를 사용하여 도면의 축척 비율을 조정합니다. 이 옵션은
실제로 도면 개체의 크기를 조정하거나 다른 식으로 수정하지 않으며, 도면을
종이에 맞추는 데 적용되는 축척 비율만 설정합니다.  
축척 콤보 상자 오른쪽의 버튼을 사용하여 선택한 종이에 도면을 자동으로
맞추거나 페이지를 자동으로 추가하여 전체 도면을 여러 페이지에 인쇄할 수
있습니다.  
„종이 위치 이동” 도구를 선택한 후 마우스 왼쪽 버튼으로 끌어 종이를
자유롭게 이동할 수 있습니다. 마우스 오른쪽 버튼을 클릭하거나 Escape 키를
눌러 도구를 종료합니다.  
옵션 도구 모음은 도면 환경 설정 대화 상자에서도 액세스할 수 있는 몇 가지
일반적인 도면 환경 설정을 빠르게 변경하는 다른 도구도 제공합니다.
