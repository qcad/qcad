Попередній перегляд друку показує, як виглядатиме ваше креслення на роздруківці. Використовуйте комбінований список на панелі параметрів, щоб налаштувати масштабний коефіцієнт вашого креслення. Цей параметр насправді не масштабує та не змінює ваші об'єкти креслення, він лише встановлює масштабний коефіцієнт, який застосовується для вписування вашого креслення на папір.  
Кнопки праворуч від комбінованого списку масштабу можна використовувати для автоматичного вписування вашого креслення на вибраний папір або для автоматичного додавання сторінок, щоб усе ваше креслення можна було надрукувати на кількох сторінках.  
Після вибору інструмента „Move Paper Position" ви можете вільно переміщати папір, перетягуючи його лівою кнопкою миші. Натисніть праву кнопку миші або клавішу Escape, щоб завершити інструмент.  
Панель параметрів пропонує деякі інші інструменти для швидкої зміни деяких поширених параметрів креслення, які також доступні в діалоговому вікні параметрів креслення.
