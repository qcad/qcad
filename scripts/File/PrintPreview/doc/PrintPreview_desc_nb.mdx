Forhåndsvisningen av utskrift viser hvordan tegningen din vil se ut i utskriften.
Bruk kombinasjonsboksen i verktøylinjen for alternativer til å justere
skaleringsfaktoren til tegningen din. Dette alternativet skalerer eller endrer
ikke selve tegningsobjektene dine, det setter bare opp skaleringsfaktoren som
brukes for å tilpasse tegningen din på papir.  
Knappene til høyre for skaleringskombinasjonsboksen kan brukes til automatisk å
tilpasse tegningen din til det valgte papiret eller til automatisk å legge til
sider, slik at hele tegningen din kan skrives ut på flere sider.  
Etter at du har valgt verktøyet «Flytt papirposisjon», kan du flytte papiret
fritt rundt ved å dra det med venstre museknapp. Klikk på høyre museknapp
eller trykk på escape-tasten for å avslutte verktøyet.  
Verktøylinjen for alternativer tilbyr noen andre verktøy for raskt å endre noen
vanlige tegningsinnstillinger som også er tilgjengelige i dialogen for
tegningsinnstillinger.
