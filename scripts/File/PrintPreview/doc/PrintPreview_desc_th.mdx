ตัวอย่างก่อนพิมพ์แสดงว่าแบบวาดของคุณจะมีลักษณะอย่างไรในงานพิมพ์ ใช้กล่องคอมโบในแถบเครื่องมือตัวเลือกเพื่อปรับตัวคูณสเกลของแบบวาดของคุณ ตัวเลือกนี้ไม่ได้ปรับสเกลหรือแก้ไขเอนทิตีแบบวาดของคุณจริงๆ เพียงตั้งค่าตัวคูณสเกลที่ใช้ในการทำให้แบบวาดของคุณพอดีกับกระดาษ  
ปุ่มทางด้านขวาของกล่องคอมโบสเกลสามารถใช้เพื่อทำให้แบบวาดของคุณพอดีกับกระดาษที่เลือกโดยอัตโนมัติ หรือเพื่อเพิ่มหน้าโดยอัตโนมัติ เพื่อให้แบบวาดทั้งหมดของคุณสามารถพิมพ์ได้บนหลายหน้า  
หลังจากเลือกเครื่องมือ „Move Paper Position" คุณสามารถเลื่อนกระดาษไปมาได้อย่างอิสระโดยการลากด้วยปุ่มซ้ายของเมาส์ คลิกปุ่มขวาของเมาส์หรือกดปุ่ม Escape เพื่อสิ้นสุดเครื่องมือ  
แถบเครื่องมือตัวเลือกมีเครื่องมืออื่นๆ บางอย่างสำหรับเปลี่ยนการตั้งค่าแบบวาดทั่วไปบางอย่างอย่างรวดเร็ว ซึ่งเข้าถึงได้จากกล่องโต้ตอบการตั้งค่าแบบวาดเช่นกัน
