Pratinjau cetak menunjukkan bagaimana gambar Anda akan terlihat pada hasil cetak. Gunakan kotak kombo pada bilah alat opsi untuk menyesuaikan faktor skala gambar Anda. Opsi ini tidak benar-benar menskalakan atau memodifikasi entitas gambar Anda, opsi ini hanya menyiapkan faktor skala yang diterapkan untuk menyesuaikan gambar Anda pada kertas.  
Tombol di sebelah kanan kotak kombo skala dapat digunakan untuk secara otomatis menyesuaikan gambar Anda pada kertas yang dipilih atau untuk secara otomatis menambahkan halaman, sehingga seluruh gambar Anda dapat dicetak pada beberapa halaman.  
Setelah memilih alat "Move Paper Position", Anda dapat memindahkan kertas secara bebas dengan menyeretnya menggunakan tombol kiri mouse. Klik tombol kanan mouse atau tekan tombol escape untuk mengakhiri alat.  
Bilah alat opsi menawarkan beberapa alat lain untuk dengan cepat mengubah beberapa preferensi gambar umum yang juga dapat diakses dalam dialog preferensi gambar.
