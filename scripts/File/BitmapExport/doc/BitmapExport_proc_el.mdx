Εμφανίζεται ένας διάλογος για την επιλογή του αρχείου εξόδου. Αφού οριστεί το όνομα του αρχείου,
ένας δεύτερος διάλογος σάς ζητά το μέγεθος του bitmap και το προτιμώμενο
χρώμα φόντου.  
Σημειώστε ότι η εξαγωγή σε bitmap με πολύ μεγάλο μέγεθος μπορεί να διαρκέσει πολύ
χρόνο, ανάλογα με το υλικό σας. Το μέγιστο μέγεθος για τα bitmap είναι πλάτος x
ύψος ≤ 2.147.483.647.
