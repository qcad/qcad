In einem Dialog kann die zu erstellende Datei bestimmt werden. Nach der Auswahl des Dateinamens wird ein zweiter Dialog mit Optionen für den Export angezeigt. Hier können Sie die Größe des Bitmaps in Pixeln bestimmen und zwischen den Hintergrundfarben schwarz oder weiß wählen.
Hinweis: Der Export von Bitmaps mit sehr großer Größe kann je nach Hardware sehr lange dauern. Die maximale Größe für Bitmaps ist Breite x Höhe ≤ 2'147'483'647.
