Zobrazí se dialog pro výběr výstupního souboru. Po zadání názvu souboru
druhý dialog požádá o velikost bitmapy a preferovanou
barvu pozadí.  
Mějte na paměti, že export do bitmap s velmi velkou velikostí může trvat dlouho,
v závislosti na vašem hardwaru. Maximální velikost bitmap je šířka x
výška ≤ 2.147.483.647.
