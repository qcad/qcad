Näytetään valintaikkuna tulostiedoston valitsemiseen. Kun tiedostonimi on määritetty, toinen valintaikkuna kysyy bittikartan kokoa ja haluttua taustaväriä.  
Huomaa, että vienti erittäin suurikokoisiin bittikarttoihin voi kestää kauan laitteistostasi riippuen. Bittikarttojen enimmäiskoko on leveys x korkeus ≤ 2.147.483.647.
