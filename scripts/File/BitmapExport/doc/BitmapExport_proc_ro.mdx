Se prezintă o fereastră de dialog pentru selectarea fișierului de ieșire. După ce numele fișierului a fost specificat, o a doua fereastră de dialog vă întreabă dimensiunea bitmapului și culoarea de fundal preferată.  
Rețineți că exportul către bitmapuri cu o dimensiune foarte mare poate dura mult timp, în funcție de hardware-ul dumneavoastră. Dimensiunea maximă pentru bitmapuri este lățime x înălțime ≤ 2.147.483.647.
