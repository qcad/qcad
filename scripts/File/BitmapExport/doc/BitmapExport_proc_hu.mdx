Megjelenik egy párbeszédablak a kimeneti fájl kiválasztásához. A fájlnév megadása
után egy második párbeszédablak a bitkép méretét és a kívánt
háttérszínt kéri.  
Ne feledje, hogy a nagyon nagy méretű bitképekbe való exportálás a hardvertől
függően sokáig tarthat. A bitképek maximális mérete szélesség x
magasság ≤ 2.147.483.647.
