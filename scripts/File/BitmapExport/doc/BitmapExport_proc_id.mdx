Dialog untuk memilih file keluaran ditampilkan. Setelah nama file ditentukan, dialog kedua menanyakan ukuran bitmap dan warna latar belakang yang disukai.  
Perhatikan bahwa mengekspor ke bitmap dengan ukuran yang sangat besar dapat memakan waktu lama, tergantung perangkat keras Anda. Ukuran maksimum untuk bitmap adalah lebar x tinggi ≤ 2.147.483.647.
