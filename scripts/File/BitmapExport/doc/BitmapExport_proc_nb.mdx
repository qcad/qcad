En dialog for å velge utdatafilen vises. Etter at filnavnet er
angitt, ber en andre dialog deg om bitmapstørrelsen og den foretrukne
bakgrunnsfargen.  
Merk at eksport til bitmaps med svært stor størrelse kan ta lang
tid, avhengig av maskinvaren din. Den maksimale størrelsen for bitmaps er bredde x
høyde ≤ 2.147.483.647.
