Ця команда зберігає поточне креслення в той самий файл, з якого воно було завантажено. Якщо ви хочете зберегти новостворене креслення або зберегти поточне креслення в новий файл, використовуйте натомість меню [Файл - Зберегти як](/uk/manual/scripts/File/SaveAs/doc/SaveAs). Тоді вас попросять указати назву файлу перед збереженням креслення.
