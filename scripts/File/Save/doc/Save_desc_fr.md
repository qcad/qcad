 Cette commande enregistre le dessin en cours dans le même fichier à partir
duquel il a été chargé. Si vous voulez enregistrer un dessin nouvellement créé
ou enregistrer le dessin actuel dans un nouveau fichier, utilisez le menu [Fichier - Enregistrer sous](../../SaveAs/doc/SaveAs_fr.html). Un
nom de fichier vous sera alors demandé avant l'enregistrement du dessin.
