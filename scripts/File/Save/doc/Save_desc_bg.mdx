 Тази команда записва текущия чертеж в същия файл, от който е бил зареден. Ако
искате да запазите новосъздаден чертеж или да запазите текущия чертеж в нов
файл, използвайте вместо това менюто [File - Save As](/bg/manual/scripts/File/SaveAs/doc/SaveAs). Тогава ще бъдете
попитани за име на файла, преди чертежът да бъде запазен
