Tento príkaz uloží aktuálny výkres do rovnakého súboru, z ktorého bol načítaný. Ak chcete uložiť novovytvorený výkres alebo uložiť aktuálny výkres do nového súboru, použite namiesto toho ponuku [File - Save As](/sk/manual/scripts/File/SaveAs/doc/SaveAs). Potom budete pred uložením výkresu požiadaní o názov súboru.
