 Este comando guarda el dibujo actual en el mismo archivo desde el que se
cargó. Si desea guardar un dibujo recién creado o guardar el dibujo actual en
un nuevo archivo, utilice el menú [Archivo - Guardar como](/es/manual/scripts/File/SaveAs/doc/SaveAs). Se le
pedirá un nombre de archivo antes de guardar el dibujo.
