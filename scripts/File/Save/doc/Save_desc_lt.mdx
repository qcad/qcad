Ši komanda įrašo dabartinį brėžinį į tą patį failą, iš kurio jis buvo įkeltas. Jei norite įrašyti naujai sukurtą brėžinį arba įrašyti dabartinį brėžinį į naują failą, vietoj to naudokite meniu [File - Save As](/lt/manual/scripts/File/SaveAs/doc/SaveAs). Tada jūsų bus paklausta failo pavadinimo prieš įrašant brėžinį.
