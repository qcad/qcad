Perintah ini menyimpan gambar saat ini ke file yang sama tempat gambar dimuat. Jika Anda ingin menyimpan gambar yang baru dibuat atau menyimpan gambar saat ini ke file baru, gunakan menu [File - Save As](/id/manual/scripts/File/SaveAs/doc/SaveAs) sebagai gantinya. Anda kemudian akan diminta nama file sebelum gambar disimpan.
