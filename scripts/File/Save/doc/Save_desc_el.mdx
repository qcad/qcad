νέο αρχείο, χρησιμοποιήστε αντ' αυτού το μενού [Αρχείο - Αποθήκευση ως](/el/manual/scripts/File/SaveAs/doc/SaveAs). Τότε
θα
σας ζητηθεί ένα όνομα αρχείου πριν αποθηκευτεί το σχέδιο.
