 Este comando guarda o desenho actual no mesmo ficheiro a partir do qual foi
carregado. Se quiser guardar um desenho recentemente criado ou guardar o
desenho actual para um novo ficheiro, utilize o menu [File - Save As](/pt/manual/scripts/File/SaveAs/doc/SaveAs) em vez disso.
Ser-lhe-á então pedido um nome de ficheiro antes de o desenho ser guardado
