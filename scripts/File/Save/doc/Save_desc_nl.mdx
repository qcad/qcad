 Dit functie slaat de huidige tekening op in hetzelfde bestand waarvan het
geladen werd. Als u een nieuw gemaakte tekening wilt opslaan of de huidige
tekening in een nieuw bestand wilt opslaan, gebruikt u in plaats daarvan het
menu [Bestand - Opslaan als](/nl/manual/scripts/File/SaveAs/doc/SaveAs). Voor het
opslaan van de tekening wordt u dan om een bestandsnaam gevraagd.
