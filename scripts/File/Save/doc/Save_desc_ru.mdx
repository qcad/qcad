Эта команда сохраняет текущий чертёж в тот же файл, из которого он был
загружен. Если вы хотите сохранить вновь созданный чертёж или сохранить текущий
чертёж в новый файл, используйте вместо этого меню [Файл — Сохранить
как](/ru/manual/scripts/File/SaveAs/doc/SaveAs). Тогда перед сохранением чертежа
у вас будет запрошено имя файла.
