 Tento příkaz uloží aktuální výkres do stejného souboru, ze kterého byl načten.
Pokud chcete uložit nově vytvořený výkres nebo uložit aktuální výkres do
nového souboru, použijte místo toho nabídku [Soubor – Uložit jako](/cs/manual/scripts/File/SaveAs/doc/SaveAs). Poté
budete před uložením výkresu požádáni o název souboru.
