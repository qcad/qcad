 Ez a parancs az aktuális rajzot ugyanabba a fájlba menti, amelyből betöltötték.
Ha egy újonnan létrehozott rajzot szeretne menteni, vagy az aktuális rajzot egy
új fájlba menteni, használja helyette a [Fájl – Mentés másként](/hu/manual/scripts/File/SaveAs/doc/SaveAs) menüt. Ekkor
a rajz mentése előtt a program fájlnevet kér.
