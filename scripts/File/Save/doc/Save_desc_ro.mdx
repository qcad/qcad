Această comandă salvează desenul curent în același fișier din care a fost încărcat. Dacă doriți să salvați un desen nou creat sau să salvați desenul curent într-un fișier nou, utilizați în schimb meniul [File - Save As](/ro/manual/scripts/File/SaveAs/doc/SaveAs). Vi se va cere apoi un nume de fișier înainte de salvarea desenului.
