 Polecenie to zapisuje aktualny rysunek do tego samego pliku, z którego został
załadowany. Aby zapisać nowo utworzony rysunek lub aktualny rysunek w nowym
pliku, należy użyć menu [Plik - Zapisz jako](/pl/manual/scripts/File/SaveAs/doc/SaveAs). Zostaniesz
poproszony o nazwę pliku przed zapisaniem rysunku.
