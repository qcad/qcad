See käsk salvestab praeguse joonise samasse faili, kust see laaditi. Kui soovite salvestada äsja loodud joonise või salvestada praeguse joonise uude faili, kasutage selle asemel menüüd [File - Save As](/et/manual/scripts/File/SaveAs/doc/SaveAs). Seejärel küsitakse teilt enne joonise salvestamist failinime.
