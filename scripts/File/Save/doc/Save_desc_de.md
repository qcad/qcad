Speichert die aktuelle Zeichnung in der gleichen Datei ab von der sie
geöffnet wurde. Wenn Sie die Zeichnung stattdessen in einer neuen Datei
abspeichern möchten, benutzen Sie das Menü [Datei - Speichern unter](../../SaveAs/doc/SaveAs_de.html). Sie werden
dann vor dem Speichern nach einem neuen Dateinamen gefragt.
