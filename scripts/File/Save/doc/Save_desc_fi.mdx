Tämä komento tallentaa nykyisen piirustuksen samaan tiedostoon, josta se ladattiin. Jos haluat tallentaa juuri luodun piirustuksen tai tallentaa nykyisen piirustuksen uuteen tiedostoon, käytä sen sijaan valikkoa [File - Save As](/fi/manual/scripts/File/SaveAs/doc/SaveAs). Sinulta kysytään tällöin tiedostonimeä ennen piirustuksen tallentamista.
