 Questo comando salva il disegno corrente nello stesso file dal quale è stato
caricato. Se si desidera salvare un disegno appena creato o il disegno corrente
in un nuovo file, utilizzare il menu [File - Salva con nome](../../SaveAs/doc/SaveAs_it.html). Verrà
quindi richiesto un nome di file prima di salvare il disegno.
