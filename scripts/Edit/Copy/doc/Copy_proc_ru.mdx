1. С помощью инструментов выделения подготовьте набор объектов, которые хотите
   скопировать в буфер обмена.
2. Выберите инструмент копирования.

Стандартный инструмент копирования QCAD работает как инструмент копирования в
других приложениях. Он не позволяет выбрать базовую точку для вашего выделения,
а вместо этого автоматически вычисляет базовую точку в центре выделения. Это не
всегда желательно, поскольку вы, скорее всего, захотите разместить вставленные
объекты в точно заданном положении. Для этого QCAD предлагает альтернативный,
более специфичный для CAD инструмент копирования: [Копировать с базовой
точкой](/ru/manual/scripts/Edit/CopyWithReference/doc/CopyWithReference).

Теперь выделение находится во внутреннем буфере обмена QCAD и может быть
вставлено в тот же или другой чертёж, открытый в том же запущенном экземпляре
QCAD. Обратите внимание, что при выходе из приложения QCAD содержимое буфера
обмена теряется. Доступ к буферу обмена QCAD может получить только сам QCAD. Вы
не можете обмениваться объектами чертежа с другими приложениями через
копирование и вставку. Для этого используйте функции экспорта и импорта QCAD.
