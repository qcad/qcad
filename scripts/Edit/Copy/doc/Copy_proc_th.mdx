1. ใช้เครื่องมือเลือกเพื่อเตรียมการเลือกเอนทิตีที่คุณต้องการคัดลอกไปยังคลิปบอร์ด
2. เลือกเครื่องมือคัดลอก

เครื่องมือคัดลอกมาตรฐานของ QCAD ทำงานเหมือนเครื่องมือคัดลอกของแอปพลิเคชันอื่น ไม่อนุญาตให้คุณเลือกจุดอ้างอิงสำหรับการเลือกของคุณ แต่คำนวณจุดอ้างอิงโดยอัตโนมัติที่จุดศูนย์กลางของการเลือก ซึ่งไม่พึงประสงค์เสมอไปเนื่องจากคุณมักจะต้องการวางเอนทิตีที่วางไว้ที่ตำแหน่งที่กำหนดอย่างแม่นยำ ด้วยเหตุนี้ QCAD จึงมีเครื่องมือคัดลอกทางเลือกที่เฉพาะเจาะจงสำหรับ CAD มากขึ้น: [Copy with Reference](/th/manual/scripts/Edit/CopyWithReference/doc/CopyWithReference)

ตอนนี้การเลือกอยู่ในคลิปบอร์ดภายในของ QCAD และสามารถวางลงในแบบวาดเดียวกันหรือแบบวาดอื่นที่เปิดด้วยอินสแตนซ์ที่ทำงานอยู่เดียวกันของ QCAD โปรดทราบว่าหากคุณออกจากแอปพลิเคชัน QCAD เนื้อหาของคลิปบอร์ดจะสูญหาย คลิปบอร์ดของ QCAD สามารถเข้าถึงได้โดย QCAD เท่านั้น คุณไม่สามารถแลกเปลี่ยนเอนทิตีแบบวาดกับแอปพลิเคชันอื่นผ่านการคัดลอกและวาง โปรดใช้ฟังก์ชันการส่งออกและนำเข้าของ QCAD สำหรับสิ่งนี้
