1. 使用选择工具准备要复制到剪贴板的实体选区。
2. 选择复制工具。

QCAD 的标准复制工具的作用类似于其他应用程序的复制工具。它不允许您为选区选择参考点，而是自动在选区中心计算参考点。这并不总是理想的，因为您很可能希望将粘贴的实体放置在精确定义的位置。为此，QCAD 提供了一个替代的、更特定于 CAD 的复制工具：[带参考点复制](/zh_CN/manual/scripts/Edit/CopyWithReference/doc/CopyWithReference)。

选区现在位于 QCAD 内部剪贴板上，可以粘贴到用同一运行实例的 QCAD 打开的相同或不同图形中。请注意，如果您退出 QCAD 应用程序，剪贴板的内容将丢失。QCAD 剪贴板只能由 QCAD 本身访问。您无法通过复制和粘贴与其他应用程序交换图形实体。请为此使用 QCAD 的导出和导入功能。
