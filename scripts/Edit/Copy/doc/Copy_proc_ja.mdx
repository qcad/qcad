1. クリップボードにコピーしたいエンティティの選択を準備するために選択ツールをクリックします。
2. コピー・ツールを選んでください。

QCADの標準コピー・ツールは他のアプリケーションのコピー・ツールのように機能します。  
それは、選択のために基準点を選ぶことはできませんが、その代り、選択の中心と基準点を自動的に計算します。  
正確に定義された位置にエンティティを置きたい場合、これは必ずしも望ましいとは限りません。  
この場合は、QCADの選択肢として「[基準点を指定してコピー」](/ja/manual/scripts/Edit/CopyWithReference/doc/CopyWithReference)が存在します。

コピー後、選択したものはQCAD内部クリップボード上にあり、同じインスタンスのQCADで開かれる同じあるいは異なる図面へ貼ることができます。  
QCADアプリケーションを閉じれば、クリップボードの内容が失われることに注意してください。  
QCADクリップボードはQCAD自体によってのみアクセスすることができます。  
コピー・アンド・ペーストによって他のアプリケーションで描画エンティティを交換することはできません。  
この場合QCADのエクスポート／インポート機能を使用してください。
