Dalam dialog preferensi aplikasi, Anda dapat menyesuaikan tampilan dan perilaku QCAD serta alat dan tampilan grafisnya.  
QCAD juga menyimpan beberapa pengaturan ini di dalam gambar. Pengaturan tersebut dapat diubah dalam preferensi aplikasi sebagai pengaturan default untuk gambar baru. Dengan kata lain, pengaturan tersebut tidak memengaruhi gambar yang sudah ada tetapi hanya gambar yang dibuat setelah preferensi diubah.
