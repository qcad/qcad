 Das Werkzeug zum Ausschneiden mit Referenzpunkt funktioniert wie das
Werkzeug zum [Kopieren mit Referenzpunkt](../../CopyWithReference/doc/CopyWithReference_de.html).
Der einzige Unterschied ist, dass die selektierten Objekte
aus der Zeichnung gelöscht werden nachdem sie auf die Zwischenablage
kopiert wurden.
