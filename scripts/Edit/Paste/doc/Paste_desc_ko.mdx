붙여넣기 명령은 이전에 클립보드에 복사된 개체를 삽입합니다. 붙여넣기는 한
도면에서 다른 도면으로 개체를 전송하는 데 특히
유용합니다.  
클립보드 내용에서 사용하는 모든 레이어가 현재 도면의 레이어 목록에
붙여넣어집니다. 같은 이름의 기존 레이어는 원하는 경우 덮어쓸 수
있습니다(옵션 도구 모음의 „레이어 덮어쓰기” 옵션).  
클립보드에 있는 블록 참조는 참조하는 블록 정의와 함께 붙여넣어집니다.
현재 도면의 블록은 원하는 경우 덮어쓸 수 있습니다(옵션 도구 모음의 „블록
덮어쓰기” 옵션).  
옵션 도구 모음은 붙여넣는 동안 클립보드 내용의 크기를 조정하거나
회전하거나 뒤집는 몇 가지 도구도 제공합니다.
