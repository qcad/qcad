Atur titik target untuk entitas yang ditempel dengan mouse atau masukkan koordinat pada baris perintah. Titik target sesuai dengan titik referensi yang dipilih saat menyalin atau memotong entitas jika Copy / Cut with Reference digunakan. Jika Cut atau Copy tanpa titik referensi digunakan, pusat pemilihan digunakan sebagai titik referensi.
