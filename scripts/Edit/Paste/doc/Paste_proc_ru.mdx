Задайте целевую точку для вставляемых объектов с помощью мыши или введите
координату в командной строке. Целевая точка соответствует базовой точке,
которая была выбрана при копировании или вырезании объектов, если
использовалось «Копировать / Вырезать с базовой точкой». Если использовалось
«Вырезать» или «Копировать» без базовой точки, в качестве базовой точки
используется центр выделения.
