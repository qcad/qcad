Установіть цільову точку для вставлених об'єктів за допомогою миші або введіть координату в командному рядку. Цільова точка відповідає опорній точці, яку було вибрано під час копіювання або вирізання об'єктів, якщо використовувалося Копіювати / Вирізати з опорою. Якщо використовувалося Вирізати або Копіювати без опорної точки, центр виділення використовується як опорна точка.
