Команда вставки вставляет объекты, которые ранее были скопированы в буфер
обмена. Вставка особенно полезна для переноса объектов из одного чертежа в
другой.  
Все слои, используемые содержимым буфера обмена, вставляются в список слоёв
текущего чертежа. Существующие слои с тем же именем могут быть перезаписаны при
необходимости (опция «перезаписать слои» на панели опций).  
Ссылки на блоки, находящиеся в буфере обмена, вставляются вместе с
определениями блоков, на которые они ссылаются. Блоки в текущем чертеже могут
быть перезаписаны при необходимости (опция «перезаписать блоки» на панели
опций).  
Панель опций также предлагает некоторые инструменты для масштабирования,
поворота или отражения содержимого буфера обмена при вставке.
