Αυτό το εργαλείο εναλλάσσει την κατάσταση κλειδώματος του τρέχοντος στρώματος. Μπορείτε επίσης να
εναλλάξετε την κατάσταση κλειδώματος των στρωμάτων κάνοντας κλικ στο εικονίδιο κλειδώματος δίπλα στο
όνομα του στρώματος στη λίστα στρωμάτων.

Οι οντότητες σε κλειδωμένα στρώματα δεν μπορούν να επιλεγούν, να διαγραφούν ή να τροποποιηθούν.
