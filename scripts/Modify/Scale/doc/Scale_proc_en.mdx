import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Select the entities you want to scale.
2. Launch this tool.
3. Set the center of the scaling with the mouse or enter a coordinate in the
   command line.
4. The scale dialog is displayed where you can enter the scale factor.  
   If you want to scale with two different factors in X and Y direction,
   uncheck the button for proportional scaling:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   You can then enter two different scaling factors for X and Y.  
   If you want to scale the selection by mouse, check the mouse cursor
   button:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Click "OK".
6. If you have previously chosen to scale the selection by mouse,
   you now have to specify a reference point and a target point for
   the scaling operation.
