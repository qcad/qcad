import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Odaberite entitete koje želite skalirati.
2. Pokrenite ovaj alat.
3. Postavite središte skaliranja pomoću miša ili unesite koordinatu u
   naredbenu liniju.
4. Prikazuje se dijalog mjerila u kojem možete unijeti faktor mjerila.  
   Ako želite skalirati s dvama različitim faktorima u smjeru X i Y,
   poništite oznaku gumba za proporcionalno skaliranje:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Zatim možete unijeti dva različita faktora skaliranja za X i Y.  
   Ako odabir želite skalirati mišem, označite gumb pokazivača
   miša:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Kliknite „OK”.
6. Ako ste prethodno odabrali skaliranje odabira mišem,
   sada morate odrediti referentnu točku i ciljnu točku za
   operaciju skaliranja.
