import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. 一定の比率で拡縮したいエンティティを選択します。
2. このツールを始めてください。
3. マウスでスケーリングの中心を設定するか、あるいはコマンドラインの中で座標を入力してください。
4. スケール係数を入力することができるところで、規模ダイアログが表示されます。
   XとY方向の2つの異なる倍率で拡縮したい場合は、比率を連動のボタンのチェックを外します:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   その後、XとYに2つの異なる計数逓減率を登録することができます。  
   マウスによる選択をしたい場合は、マウス・カーソル・ボタンをチェックします:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. 「OK」をクリックします。
6. マウスによる選択を計ることに以前に決めていれば、拡縮のための基準点およびターゲット点を指定しなければなりません。
