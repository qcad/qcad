import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Wybierz jednostki, które chcesz skalować.
2. Uruchom to narzędzie.
3. Środkową część skalowania ustawić myszką lub wprowadzić współrzędne w
 wierszu poleceń.
4. Zostanie wyświetlone okno dialogowe skali, w którym można wprowadzić
 współczynnik skali.  
Jeśli chcesz skalować z dwoma różnymi współczynnikami w kierunku X i Y,
 usuń zaznaczenie przycisku dla skalowania proporcjonalnego:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
Następnie można wprowadzić dwa różne współczynniki skalowania dla X i
 Y. Aby przeskalować zaznaczenie myszką, należy zaznaczyć kursor myszy.
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Kliknij "OK".
6. Jeśli wcześniej wybrano skalowanie zaznaczenia za pomocą myszy, należy
 teraz określić punkt odniesienia i punkt docelowy dla operacji
 skalowania.
