import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Vyberte prvky, jejichž měřítko chcete změnit.
2. Spusťte tento nástroj.
3. Nastavte střed změny měřítka pomocí myši nebo zadejte souřadnici do
   příkazového řádku.
4. Zobrazí se dialog měřítka, kde můžete zadat faktor měřítka.  
   Pokud chcete změnit měřítko dvěma různými faktory ve směru X a Y,
   zrušte zaškrtnutí tlačítka pro proporcionální změnu měřítka:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Poté můžete zadat dva různé faktory měřítka pro X a Y.  
   Pokud chcete změnit měřítko výběru myší, zaškrtněte tlačítko kurzoru
   myši:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Klikněte na „OK”.
6. Pokud jste si dříve zvolili změnu měřítka výběru myší,
   nyní musíte určit referenční bod a cílový bod pro
   operaci změny měřítka.
