import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Επιλέξτε τις οντότητες που θέλετε να κλιμακώσετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Ορίστε το κέντρο της κλιμάκωσης με το ποντίκι ή εισαγάγετε μια συντεταγμένη στη
   γραμμή εντολών.
4. Εμφανίζεται ο διάλογος κλίμακας όπου μπορείτε να εισαγάγετε τον συντελεστή κλίμακας.  
   Αν θέλετε να κλιμακώσετε με δύο διαφορετικούς συντελεστές στην κατεύθυνση X και Y,
   αποεπιλέξτε το κουμπί για την αναλογική κλιμάκωση:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Μπορείτε στη συνέχεια να εισαγάγετε δύο διαφορετικούς συντελεστές κλιμάκωσης για το X και το Y.  
   Αν θέλετε να κλιμακώσετε την επιλογή με το ποντίκι, επιλέξτε το κουμπί δρομέα
   ποντικιού:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Κάντε κλικ στο «OK».
6. Αν είχατε προηγουμένως επιλέξει να κλιμακώσετε την επιλογή με το ποντίκι,
   τώρα πρέπει να ορίσετε ένα σημείο αναφοράς και ένα σημείο προορισμού για
   τη λειτουργία κλιμάκωσης.
