import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Pilih entitas yang ingin Anda skalakan.
2. Jalankan alat ini.
3. Atur pusat penskalaan dengan mouse atau masukkan koordinat pada baris perintah.
4. Dialog skala ditampilkan di mana Anda dapat memasukkan faktor skala.  
   Jika Anda ingin menskalakan dengan dua faktor berbeda pada arah X dan Y, batalkan centang tombol untuk penskalaan proporsional:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Anda kemudian dapat memasukkan dua faktor penskalaan berbeda untuk X dan Y.  
   Jika Anda ingin menskalakan pemilihan dengan mouse, centang tombol kursor mouse:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Klik "OK".
6. Jika Anda sebelumnya memilih untuk menskalakan pemilihan dengan mouse, Anda sekarang harus menentukan titik referensi dan titik target untuk operasi penskalaan.
