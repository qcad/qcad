import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Seleccione as entidades que pretende escalar.
2. Lançar esta ferramenta.
3. Definir o centro da escala com o rato ou introduzir uma coordenada na
 linha de comando.
4. O diálogo da escala é exibido onde se pode introduzir o factor de escala.  
Se desejar escalar com dois factores diferentes na direcção X e Y,
 desmarque o botão para escalar proporcionalmente:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
Pode então introduzir dois factores de escala diferentes para X e Y.  
Se quiser escalar a selecção com o rato, verifique o botão do cursor do
 rato:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Clique em "OK".
6. Se optou anteriormente por escalar a selecção com o rato, tem agora de
 especificar um ponto de referência e um ponto alvo para a operação de
 escalonamento.
