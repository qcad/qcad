import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Valitse kohteet, jotka haluat skaalata.
2. Käynnistä tämä työkalu.
3. Aseta skaalauksen keskipiste hiirellä tai syötä koordinaatti komentoriville.
4. Näytetään skaalausvalintaikkuna, jossa voit syöttää mittakaavakertoimen.  
   Jos haluat skaalata kahdella eri kertoimella X- ja Y-suunnassa, poista valinta suhteellisen skaalauksen painikkeesta:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Voit sitten syöttää kaksi eri skaalauskerrointa X:lle ja Y:lle.  
   Jos haluat skaalata valinnan hiirellä, valitse hiiren osoitin -painike:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Napsauta "OK".
6. Jos valitsit aiemmin valinnan skaalaamisen hiirellä, sinun täytyy nyt määrittää skaalaustoiminnon viitepiste ja kohdepiste.
