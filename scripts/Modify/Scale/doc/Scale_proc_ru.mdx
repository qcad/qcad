import KeepProportionsOn from './KeepProportionsOn.png';
import ByMouse from './ByMouse.png';

1. Выберите объекты, которые хотите масштабировать.
2. Запустите этот инструмент.
3. Задайте центр масштабирования с помощью мыши или введите координату в
   командной строке.
4. Отображается диалог масштабирования, в котором вы можете ввести масштабный
   коэффициент.  
   Если вы хотите масштабировать с двумя разными коэффициентами по направлениям
   X и Y, снимите флажок кнопки пропорционального масштабирования:  
   <Image src={KeepProportionsOn} alt="KeepProportionsOn" width={21} height={40} />  
   Затем вы можете ввести два разных масштабных коэффициента для X и Y.  
   Если вы хотите масштабировать выделение мышью, установите кнопку курсора
   мыши:  
   <Image src={ByMouse} alt="ByMouse" width={21} height={40} />
5. Нажмите «OK».
6. Если вы ранее выбрали масштабирование выделения мышью, теперь вам нужно
   указать опорную точку и целевую точку для операции масштабирования.
