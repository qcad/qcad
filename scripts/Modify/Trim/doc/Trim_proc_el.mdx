1. Επιλέξτε την οριοθετική οντότητα προς την οποία θέλετε να κόψετε ή να επεκτείνετε μία ή
   περισσότερες άλλες οντότητες.
2. Επιλέξτε τις οντότητες που θέλετε να κόψετε προς την οριοθετική οντότητα. Συχνά
   υπάρχουν δύο δυνατότητες για την οντότητα που θα κοπεί. Στο παρακάτω
   παράδειγμα, ίσως θέλετε το επάνω μέρος της γραμμής να παραμείνει στο σχέδιο και
   το κάτω μέρος να κοπεί. Σε αυτή την περίπτωση θα έπρεπε να κάνετε κλικ στην
   οντότητα κοπής στο επάνω μέρος. Κάντε πάντα κλικ στο μέρος της οντότητας που
   θέλετε να διατηρήσετε.  

   <Image src={TrimFigure} alt="TrimFigure" width={223} height={90} />

3. Κάντε δεξί κλικ δύο φορές ή πατήστε Escape δύο φορές για να τερματίσετε το εργαλείο.
