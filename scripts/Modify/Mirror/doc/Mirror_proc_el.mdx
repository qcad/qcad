1. Επιλέξτε τις οντότητες που θέλετε να καθρεφτίσετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Ορίστε ένα πρώτο σημείο πάνω στον άξονα καθρεφτισμού με το ποντίκι ή εισαγάγετε μια
   συντεταγμένη στη γραμμή εντολών.
4. Ορίστε ένα δεύτερο σημείο πάνω στον άξονα καθρεφτισμού.
5. Εμφανίζεται ο διάλογος καθρεφτισμού.  
   Για να καθρεφτίσετε τις οντότητες χωρίς να διατηρήσετε τις αρχικές οντότητες, επιλέξτε
   «Διαγραφή πρωτοτύπου», για να τις αντιγράψετε επιλέξτε «Διατήρηση πρωτοτύπου».  
   Οι νέες οντότητες τοποθετούνται στο ίδιο στρώμα με τις αρχικές και έχουν
   τα ίδια χαρακτηριστικά. Για να χρησιμοποιήσετε αντ' αυτού το τρέχον στρώμα και τα τρέχοντα χαρακτηριστικά,
   επιλέξτε «Χρήση τρέχοντος στρώματος και χαρακτηριστικών».
6. Κάντε κλικ στο «OK» για να καθρεφτίσετε τις οντότητες.
