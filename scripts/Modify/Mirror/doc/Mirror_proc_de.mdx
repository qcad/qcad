1. Selektieren Sie die Objekte, die gespiegelt werden sollen.
2. Starten Sie dieses Werkzeug.
3. Bestimmen Sie einen Punkt auf der Spiegelachse mit der Maus oder indem
   Sie eine Koordinate in die Kommandozeile eingeben.
4. Bestimmen Sie einen zweiten Punkt auf der Spiegelachse.
5. Der Dialog mit den Optionen zum Spiegeln von Objekten wird angezeigt.  
   Um die selektierten Objekte zu spiegeln ohne die originalen Objekte
   beizubehalten, aktivieren Sie "Original löschen", um sie zu kopieren
   "Original beibehalten".  
   Die neuen Objekte werden auf denselben Layer plaziert wie die Originale
   und haben auch dieselben Attribute. Um stattdessen den aktuellen Layer und
   die aktuellen Attribute zu verwenden, aktivieren Sie die Option "Aktueller
   Layer / Attribute".
6. Klicken Sie "OK" um die Objekte zu spiegeln.
