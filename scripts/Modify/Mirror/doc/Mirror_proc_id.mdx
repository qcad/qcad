1. Pilih entitas yang ingin Anda cerminkan.
2. Jalankan alat ini.
3. Tentukan titik pertama pada sumbu cermin dengan mouse atau masukkan koordinat pada baris perintah.
4. Tentukan titik kedua pada sumbu cermin.
5. Dialog cermin ditampilkan.  
   Untuk mencerminkan entitas tanpa mempertahankan entitas asli, pilih "Delete Original", untuk menyalinnya pilih "Keep Original".  
   Entitas baru ditempatkan pada lapisan yang sama dengan aslinya dan memiliki atribut yang sama. Untuk menggunakan lapisan saat ini dan atribut saat ini sebagai gantinya, centang "Use current layer and attributes".
6. Klik "OK" untuk mencerminkan entitas.
