1. Sélectionnez les entités que vous voulez mettre en miroir.
2. Lancez cet outil.
3. Indiquez un premier point sur l'axe du miroir avec la souris ou entrez
 une coordonnée dans la ligne de commande.
4. Spécifiez un deuxième point sur l'axe du miroir.
5. La boîte de dialogue miroir s'affiche.  
Pour dupliquer les entités sans conserver les entités originales,
 choisissez "Supprimer l'original", pour les copier choisissez "Conserver
 l'original".  
Les nouvelles entités sont placées sur le même calque que les originaux
 et ont les mêmes attributs. Pour utiliser le calque courant et les attributs
 courants à la place, cochez "Utiliser le calque courant et les
 attributs".
6. Cliquez sur "OK" pour faire un miroir des entités.
