1. Κάντε κλικ στην οντότητα κειμένου που θέλετε να επεξεργαστείτε.
2. Εμφανίζεται ο διάλογος κειμένου όπου μπορείτε να προσαρμόσετε το κείμενο και τα
   χαρακτηριστικά του.

Σημειώστε ότι αντί να χρησιμοποιήσετε αυτό το εργαλείο, μπορείτε επίσης να κάνετε διπλό κλικ στην οντότητα
 κειμένου που θέλετε να επεξεργαστείτε.
