1. Pilih entitas yang ingin Anda pecah.
2. Klik titik pertama di mana entitas akan dibagi.
3. Jika titik tidak terletak pada entitas, titik pemisah diasumsikan sebagai titik terdekat pada entitas.
4. Klik titik kedua di mana entitas akan dibagi.
5. Segmen antara dua titik dihapus jika opsi "Remove Segment" pada bilah alat opsi aktif. Jika tidak, entitas hanya dipisahkan pada titik-titik yang diberikan.
