1. Выберите объект, который хотите разбить.
2. Щёлкните первую точку, в которой объект должен быть разделён.
3. Если точка не лежит на объекте, разделяющей точкой считается ближайшая точка
   на объекте.
4. Щёлкните вторую точку, в которой объект должен быть разделён.
5. Сегмент между двумя точками удаляется, если активна опция «Удалить сегмент»
   на панели опций. В противном случае объект только разделяется в заданных
   точках.
