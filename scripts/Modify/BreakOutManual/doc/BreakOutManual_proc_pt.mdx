1. Selecione a entidade que pretende dividir.
2. Clique no primeiro ponto onde a entidade deve ser dividida.
3. Se o ponto não estiver sobre a entidade, assume-se que o ponto de separação é o ponto mais próximo na entidade.
4. Clique no segundo ponto onde a entidade deve ser dividida.
5. O segmento entre os dois pontos é removido se a opção "Remover segmento" na barra de opções estiver ativa. Caso contrário, a entidade é apenas separada nos pontos indicados.
