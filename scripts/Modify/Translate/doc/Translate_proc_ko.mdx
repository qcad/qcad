1. 이동하거나 복사할 개체를 선택합니다.
2. 이 도구를 시작합니다.
3. 마우스로 기준점을 설정하거나 명령줄에 좌표를
   입력합니다.
4. 대상 점을 설정합니다. 선택한 개체를 주어진 양만큼 이동하려면 상대
   좌표를 입력합니다. 예를 들어 오른쪽으로 도면 단위 50만큼 이동하려면
   명령줄에 @50,0을 입력합니다.
5. 이동 대화 상자가 표시됩니다.  
   개체를 이동하려면 „원본 삭제”를 선택하고, 복사하려면 „원본 유지”를
   선택합니다. „다중 복사”를 선택하고 아래 텍스트 줄에 복사 개수를
   입력하여 임의의 개수의 복사본을 한 번에 만들 수도 있습니다. „9”는
   9개의 복사본을 만들고 원본을 유지하므로 결국 선택한 개체의 인스턴스가
   10개가 된다는 점에 유의하십시오.  
   새 개체는 원본과 같은 레이어에 배치되며 같은 속성을 갖습니다. 대신
   현재 레이어와 현재 속성을 사용하려면 „현재 레이어 및 속성 사용”을
   선택합니다.
6. „OK”를 클릭하여 개체를 이동하거나 복사합니다.
