1. 移動かコピーしたいエンティティを選択します。
2. このツールを始めてください。
3. マウスで基準点を設定するか、あるいはコマンドラインの中で座標を入力してください。
4. ターゲット点をセットします。  
   与えられた量によって選択されたエンティティを移動させるためには、相対座標に入ってください。  
   例えば、右にそれを50ユニット移動させるために、コマンドラインで@50,0を入力します。
5. 移動ダイアログが表示されます。  
   エンティティを移動させるためには、「オリジナルを削除」に、それらをコピーするためには「オリジナルを維持」に設定します。  
   さらに、「多重コピー」を選びテキスト・ラインにコピーの数を下に入力することにより、コピーの所定の数を一度に作成することができます。  
   「9」が9つのコピーを作成しオリジナルを維持するだろうということに注意します。したがって、その場合選択されたエンティティの10の実体が存在します。  
   新しいエンティティはオリジナルと同じ画層に置かれ、同じ属性があります。  
   現在の画層および現在の属性を代わりに使用するためには、「現在の画層および属性を使用」をチェックしてください。
6. エンティティを移動かコピーするために「OK」をクリックします。
