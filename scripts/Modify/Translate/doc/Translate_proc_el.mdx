1. Επιλέξτε τις οντότητες που θέλετε να μετακινήσετε ή να αντιγράψετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Ορίστε το σημείο αναφοράς με το ποντίκι ή εισαγάγετε μια συντεταγμένη στη
   γραμμή εντολών.
4. Ορίστε το σημείο προορισμού. Για να μετακινήσετε τις επιλεγμένες οντότητες κατά ένα δοθέν ποσό,
   εισαγάγετε μια σχετική συντεταγμένη. Για παράδειγμα, για να τις μετακινήσετε κατά 50 μονάδες σχεδίου προς
   τα δεξιά, εισαγάγετε @50,0 στη γραμμή εντολών.
5. Εμφανίζεται ο διάλογος μετακίνησης.  
   Για να μετακινήσετε τις οντότητες, επιλέξτε «Διαγραφή πρωτοτύπου», για να τις αντιγράψετε επιλέξτε
   «Διατήρηση πρωτοτύπου». Μπορείτε επίσης να δημιουργήσετε οποιονδήποτε αριθμό αντιγράφων ταυτόχρονα
   επιλέγοντας «Πολλαπλά αντίγραφα» και εισάγοντας τον αριθμό των αντιγράφων στη γραμμή κειμένου
   παρακάτω. Σημειώστε ότι το «9» θα δημιουργήσει 9 αντίγραφα και θα διατηρήσει το πρωτότυπο - οπότε θα
   υπάρχουν 10 στιγμιότυπα των επιλεγμένων οντοτήτων στο τέλος.  
   Οι νέες οντότητες τοποθετούνται στο ίδιο στρώμα με τις αρχικές και έχουν
   τα ίδια χαρακτηριστικά. Για να χρησιμοποιήσετε αντ' αυτού το τρέχον στρώμα και τα τρέχοντα χαρακτηριστικά,
   επιλέξτε «Χρήση τρέχοντος στρώματος και χαρακτηριστικών».
6. Κάντε κλικ στο «OK» για να μετακινήσετε ή να αντιγράψετε τις οντότητες.
