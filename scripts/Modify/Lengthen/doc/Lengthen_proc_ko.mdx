1. 옵션 도구 모음에 개체를 늘이려는 거리를 입력합니다. 양수 값은
   개체를 늘이고 음수 값은 줄입니다. 예를 들어 „5” 값은 선택한 개체를
   5단위만큼
   늘입니다.
2. 수정하려는 끝 근처에서 늘이거나 줄일 개체를
   선택합니다.
3. 마우스 오른쪽 버튼을 클릭하거나 Escape 키를 눌러 도구를 종료합니다.
