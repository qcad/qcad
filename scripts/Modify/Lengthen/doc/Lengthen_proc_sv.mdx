1. Ange det avstånd som du vill förlänga en enhet med i verktygsfältet för
   alternativ. Ett positivt värde förlänger enheten, medan ett negativt värde
   förkortar den. Ett värde på "5" förlänger t.ex. den valda enheten med 5
   enheter.
2. Välj den enhet som ska förlängas eller förkortas någonstans nära den
   ände som du vill ändra.
3. Högerklicka eller tryck på Escape-tangenten för att avsluta
   verktyget.
