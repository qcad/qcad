1. Εισαγάγετε την απόσταση κατά την οποία θέλετε να επιμηκύνετε μια οντότητα στη γραμμή
   εργαλείων επιλογών. Μια θετική τιμή επιμηκύνει την οντότητα, ενώ μια αρνητική τιμή
   τη βραχύνει. Π.χ. μια τιμή «5» θα επιμηκύνει την επιλεγμένη οντότητα κατά 5
   μονάδες.
2. Επιλέξτε την οντότητα που θα επιμηκυνθεί ή θα βραχυνθεί κάπου κοντά στο
   άκρο που θέλετε να τροποποιήσετε.
3. Κάντε δεξί κλικ ή πατήστε το πλήκτρο Escape για να τερματίσετε το εργαλείο.
