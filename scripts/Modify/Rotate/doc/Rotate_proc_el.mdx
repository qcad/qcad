import ByMouse from './ByMouse.png';

1. Επιλέξτε τις οντότητες που θέλετε να περιστρέψετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Ορίστε το κέντρο της περιστροφής με το ποντίκι ή εισαγάγετε μια συντεταγμένη στη
   γραμμή εντολών.
4. Εμφανίζεται ο διάλογος περιστροφής όπου μπορείτε να εισαγάγετε τον αριθμό των αντιγράφων
   και τη γωνία περιστροφής.  
   Αν θέλετε να ορίσετε τη γωνία περιστροφής με τον δρομέα του ποντικιού, επιλέξτε
   το κουμπί δρομέα ποντικιού που εμφανίζεται δίπλα στο πεδίο εισαγωγής γωνίας:  
   <Image src={ByMouse} alt="ByMouse" width={40} height={40} />  
   Για να διαγράψετε τις αρχικές οντότητες, επιλέξτε «Διαγραφή πρωτοτύπου», για να τις αντιγράψετε
   επιλέξτε «Διατήρηση πρωτοτύπου». Μπορείτε επίσης να δημιουργήσετε οποιονδήποτε αριθμό αντιγράφων
   επιλέγοντας «Πολλαπλά αντίγραφα».  
   Οι νέες οντότητες τοποθετούνται στο ίδιο στρώμα με τις αρχικές και
   έχουν τα ίδια χαρακτηριστικά. Για να χρησιμοποιήσετε αντ' αυτού το τρέχον στρώμα και τα τρέχοντα
   χαρακτηριστικά, επιλέξτε «Χρήση τρέχοντος στρώματος και χαρακτηριστικών».
5. Κάντε κλικ στο «OK».
6. Αν είχατε προηγουμένως επιλέξει να ορίσετε τη γωνία περιστροφής με το ποντίκι,
   τώρα πρέπει πρώτα να ορίσετε ένα σημείο αναφοράς για την περιστροφή και μετά ένα
   σημείο προορισμού. Η γωνία περιστροφής είναι η γωνία που περικλείεται από το σημείο
   αναφοράς, το κέντρο περιστροφής και το σημείο προορισμού.
