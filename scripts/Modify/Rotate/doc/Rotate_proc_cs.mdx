import ByMouse from './ByMouse.png';

1. Vyberte prvky, které chcete otočit.
2. Spusťte tento nástroj.
3. Nastavte střed otočení pomocí myši nebo zadejte souřadnici do
   příkazového řádku.
4. Zobrazí se dialog otočení, kde můžete zadat počet kopií a
   úhel otočení.  
   Pokud chcete úhel otočení určit kurzorem myši, zaškrtněte
   tlačítko kurzoru myši, které se zobrazuje vedle pole úhlu:  
   <Image src={ByMouse} alt="ByMouse" width={40} height={40} />  
   Pro odstranění původních prvků zaškrtněte „Odstranit originál”, pro jejich zkopírování
   vyberte „Zachovat originál”. Můžete také vytvořit libovolný počet kopií
   výběrem „Více kopií”.  
   Nové prvky se umístí na stejnou vrstvu jako originály a
   mají stejné atributy. Chcete-li místo toho použít aktuální vrstvu a aktuální
   atributy, zaškrtněte „Použít aktuální vrstvu a atributy”.
5. Klikněte na „OK”.
6. Pokud jste si dříve zvolili určení úhlu otočení myší,
   nyní musíte nejprve určit referenční bod pro otočení a poté
   cílový bod. Úhel otočení je úhel sevřený referenčním
   bodem, středem otočení a cílovým bodem.
