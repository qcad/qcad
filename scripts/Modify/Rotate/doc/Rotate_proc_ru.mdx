import ByMouse from './ByMouse.png';

1. Выберите объекты, которые хотите повернуть.
2. Запустите этот инструмент.
3. Задайте центр поворота с помощью мыши или введите координату в командной
   строке.
4. Отображается диалог поворота, в котором вы можете ввести количество копий и
   угол поворота.  
   Если вы хотите задать угол поворота с помощью курсора мыши, установите кнопку
   курсора мыши, отображаемую рядом с полем ввода угла:  
   <Image src={ByMouse} alt="ByMouse" width={40} height={40} />  
   Чтобы удалить исходные объекты, установите «Удалить оригинал», чтобы
   скопировать их — выберите «Сохранить оригинал». Вы также можете создать любое
   заданное количество копий, выбрав «Несколько копий».  
   Новые объекты размещаются на том же слое, что и оригиналы, и имеют те же
   атрибуты. Чтобы вместо этого использовать текущий слой и текущие атрибуты,
   установите флажок «Использовать текущий слой и атрибуты».
5. Нажмите «OK».
6. Если вы ранее выбрали задание угла поворота мышью, теперь вам нужно сначала
   указать опорную точку для поворота, а затем целевую точку. Угол поворота — это
   угол, заключённый между опорной точкой, центром поворота и целевой точкой.
