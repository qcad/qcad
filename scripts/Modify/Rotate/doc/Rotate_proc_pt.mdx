import ByMouse from './ByMouse.png';

1. Seleccione as entidades que pretende rotacionar.
2. Lançar esta ferramenta.
3. Defina o centro da rotação com o rato ou introduza uma coordenada na
   linha de comando.
4. O diálogo de rotação é exibido onde se pode introduzir o número de cópias
   e o ângulo de rotação.  
   Se desejar especificar o ângulo de rotação com o cursor do rato,
   verifique o botão do cursor do rato que é exibido ao lado da entrada do
   ângulo:  
   <Image src={ByMouse} alt="ByMouse" width={40} height={40} />  
   Para apagar as entidades originais, marcar "Apagar Original", para as
   copiar escolher "Guardar Original". Também pode criar qualquer número dado de
   cópias escolhendo "Cópias Múltiplas".  
   As novas entidades são colocadas na mesma camada que os originais e têm
   os mesmos atributos. Para utilizar a camada actual e os atributos actuais,
   assinale "Utilizar camada actual e atributos actuais".
5. Clique em "OK".
6. Se escolheu anteriormente especificar o ângulo de rotação com o rato, tem
   agora de especificar primeiro um ponto de referência para a rotação e depois
   um ponto alvo. O ângulo de rotação é o ângulo delimitado pelo ponto de
   referência, centro de rotação e ponto-alvo.
