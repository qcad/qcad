import ByMouse from './ByMouse.png';

1. Pilih entitas yang ingin Anda putar.
2. Jalankan alat ini.
3. Atur pusat rotasi dengan mouse atau masukkan koordinat pada baris perintah.
4. Dialog putar ditampilkan di mana Anda dapat memasukkan jumlah salinan dan sudut rotasi.  
   Jika Anda ingin menentukan sudut rotasi dengan kursor mouse, centang tombol kursor mouse yang ditampilkan di samping input sudut:  
   <Image src={ByMouse} alt="ByMouse" width={40} height={40} />  
   Untuk menghapus entitas asli, centang "Delete Original", untuk menyalinnya pilih "Keep Original". Anda juga dapat membuat sejumlah salinan tertentu dengan memilih "Multiple Copies".  
   Entitas baru ditempatkan pada lapisan yang sama dengan aslinya dan memiliki atribut yang sama. Untuk menggunakan lapisan saat ini dan atribut saat ini sebagai gantinya, centang "Use current layer and attributes".
5. Klik "OK".
6. Jika Anda sebelumnya memilih untuk menentukan sudut rotasi dengan mouse, Anda sekarang harus terlebih dahulu menentukan titik referensi untuk rotasi dan kemudian titik target. Sudut rotasi adalah sudut yang diapit oleh titik referensi, pusat rotasi, dan titik target.
