1. Geben Sie den Radius der Rundung in der Optionen Werkzeugleiste ein.
  2. Aktivieren Sie "Trimmen" wenn Sie die Kanten automatisch zur Rundung trimmen wollen. Wenn die Option deaktiviert ist, bleiben die Kanten unverändert.
  3. Wählen Sie die erste Kante (Linie oder Kreisbogen).
  4. Wählen Sie die zweite Kante.
