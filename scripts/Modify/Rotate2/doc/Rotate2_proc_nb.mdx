1. Velg objektene du vil rotere og motrotere.
2. Start dette verktøyet.
3. Sett senteret for hovedrotasjonen med musen eller skriv inn en koordinat
   i kommandolinjen.
4. Sett senteret for rotasjonen av de individuelle objektene. Dette andre
   rotasjonssenteret roteres sammen med objektene rundt det første
   senteret.
5. Dialogen for dobbel rotasjon vises der du kan skrive inn rotasjonsvinkelen
   og motrotasjonsvinkelen.  
   For å slette de opprinnelige objektene, merk av for «Slett original», for å kopiere dem
   velg «Behold original». Du kan også opprette et hvilket som helst antall kopier ved
   å velge «Flere kopier».  
   De nye objektene plasseres på det samme laget som originalene og
   har de samme attributtene. For å bruke det gjeldende laget og de gjeldende attributtene
   i stedet, merk av for «Bruk gjeldende lag og attributter».
6. Klikk «OK» for å rotere objektene.
