1. Välj de enheter som du vill rotera och motrotera.
2. Starta det här verktyget.
3. Ange centrum för huvudrotationen med musen eller ange en koordinat på
 kommandoraden.
4. Ställ in centrum för rotationen för de enskilda objekten. Detta andra
 centrum för rotationen roteras tillsammans med entiteterna runt det första
 centrumet.
5. Dialogrutan Rotate two visas där du kan ange rotationsvinkeln och
 motrotationsvinkeln.  
För att radera de ursprungliga entiteterna markerar du "Delete
 Original", för att kopiera dem väljer du "Keep Original". Du kan också
 skapa ett valfritt antal kopior genom att välja "Multiple Copies".  
De nya enheterna placeras på samma lager som originalen och har samma
 attribut. Om du vill använda det aktuella lagret och de aktuella attributen
 i stället kryssar du i "Use current layer and attributes" (Använd aktuellt
 lager och aktuella attribut).
6. Klicka på "OK" för att rotera enheterna.
