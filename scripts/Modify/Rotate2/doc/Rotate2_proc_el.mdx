1. Επιλέξτε τις οντότητες που θέλετε να περιστρέψετε και να αντιπεριστρέψετε.
2. Εκκινήστε αυτό το εργαλείο.
3. Ορίστε το κέντρο για την κύρια περιστροφή με το ποντίκι ή εισαγάγετε μια συντεταγμένη
   στη γραμμή εντολών.
4. Ορίστε το κέντρο της περιστροφής για τα μεμονωμένα αντικείμενα. Αυτό το δεύτερο
   κέντρο περιστροφής περιστρέφεται μαζί με τις οντότητες γύρω από το πρώτο
   κέντρο.
5. Εμφανίζεται ο διάλογος διπλής περιστροφής όπου μπορείτε να εισαγάγετε τη γωνία περιστροφής
   και τη γωνία αντιπεριστροφής.  
   Για να διαγράψετε τις αρχικές οντότητες, επιλέξτε «Διαγραφή πρωτοτύπου», για να τις αντιγράψετε
   επιλέξτε «Διατήρηση πρωτοτύπου». Μπορείτε επίσης να δημιουργήσετε οποιονδήποτε αριθμό αντιγράφων
   επιλέγοντας «Πολλαπλά αντίγραφα».  
   Οι νέες οντότητες τοποθετούνται στο ίδιο στρώμα με τις αρχικές και
   έχουν τα ίδια χαρακτηριστικά. Για να χρησιμοποιήσετε αντ' αυτού το τρέχον στρώμα και τα τρέχοντα
   χαρακτηριστικά, επιλέξτε «Χρήση τρέχοντος στρώματος και χαρακτηριστικών».
6. Κάντε κλικ στο «OK» για να περιστρέψετε τις οντότητες.
