1. Syötä viisteen geometria asetustyökaluriville. "Distance 1" on
   etäisyys, joka viistousviivalla on kahden reunan (kuvitteellisesta)
   leikkauspisteestä (esimerkissä 10). "Distance 2" on sama etäisyys toiselle
   reunalle (esimerkissä 20).
2. Valitse "Trim", jos haluat typistää kohteet automaattisesti. Jos vaihtoehto
   on poistettu käytöstä, kaksi kohdetta jäävät koskemattomiksi.
3. Valitse ensimmäinen reunakohde (viiva tai kaari).
4. Valitse toinen kohde.
