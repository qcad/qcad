1. Εισαγάγετε τη γεωμετρία της λοξότμησης στη γραμμή εργαλείων επιλογών. Η «Απόσταση 1» είναι
   η απόσταση που θα έχει η γραμμή λοξότμησης από τη (φανταστική) τομή
   των δύο ακμών (10 στο εμφανιζόμενο παράδειγμα). Η «Απόσταση 2» είναι η ίδια απόσταση
   για τη δεύτερη ακμή (20 στο εμφανιζόμενο παράδειγμα).
2. Επιλέξτε «Κοπή» αν θέλετε να κόψετε τις οντότητες αυτόματα. Αν η
   επιλογή είναι απενεργοποιημένη, οι δύο οντότητες θα παραμείνουν ανέπαφες.
3. Επιλέξτε την πρώτη οντότητα ακμής (μια γραμμή ή ένα τόξο).
4. Επιλέξτε τη δεύτερη οντότητα.
